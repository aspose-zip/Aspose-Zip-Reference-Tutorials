## Perguntas Frequentes

**Q: Can I use Aspose.Zip to create a zip archive .net on .NET Core?**  
A: Absolutamente. Aspose.Zip oferece suporte total ao .NET Core 3.1 e posteriores, bem como ao .NET 5/6/7.

**Q: How do I modify an existing archive without extracting it first?**  
A: Use as APIs “modify zip file c#” – você pode adicionar, excluir ou renomear entradas diretamente no stream do arquivo.

**Q: Which compression method gives the best ratio for text files?**  
A: Geralmente, Bzip2 ou LZMA proporcionam maior compressão para textos, porém são mais lentos que Deflate. Teste ambos para encontrar o equilíbrio ideal.

**Q: Is there a way to encrypt files while creating the zip archive?**  
A: Sim, o Aspose.Zip suporta criptografia AES. Você pode definir uma senha no objeto ZipArchive antes de salvar, habilitando **zip file encryption c#**.

**Q: Do I need to worry about file name encoding?**  
A: O Aspose.Zip lida automaticamente com nomes de arquivos Unicode, mas você pode definir explicitamente a codificação se precisar de compatibilidade com sistemas legados.
