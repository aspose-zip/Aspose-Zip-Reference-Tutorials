## よくある質問

**Q:** Aspose.Zip を使用して .NET Core 上で .net の zip アーカイブを作成できますか？  
**A:** もちろんです。Aspose.Zip は .NET Core 3.1 以降、さらに .NET 5/6/7 を完全にサポートしています。

**Q:** アーカイブを展開せずに既存のアーカイブを変更するにはどうすればよいですか？  
**A:** “modify zip file c#” API を使用します。エントリの追加、削除、名前変更をアーカイブストリーム上で直接行えます。

**Q:** テキストファイルに最適な圧縮率を提供する方式はどれですか？  
**A:** 一般的に Bzip2 または LZMA がテキストに対して高い圧縮率を示しますが、Deflate よりも遅くなります。両方をテストして最適なバランスを見つけてください。

**Q:** Zip アーカイブ作成時にファイルを暗号化する方法はありますか？  
**A:** はい、Aspose.Zip は AES 暗号化をサポートしています。保存前に ZipArchive オブジェクトにパスワードを設定すれば、**zip file encryption c#** が有効になります。

**Q:** ファイル名のエンコーディングを気にする必要がありますか？  
**A:** Aspose.Zip は Unicode ファイル名を自動的に処理しますが、レガシーシステムとの互換性が必要な場合はエンコーディングを明示的に設定できます。
