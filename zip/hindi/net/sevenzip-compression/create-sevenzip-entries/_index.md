## c# create 7z archive – चरण‑दर‑चरण गाइड

### चरण 1: रिसोर्स डायरेक्टरी पाथ सेट करें

SevenZip एंट्रीज़ बनाने से पहले, अपनी रिसोर्स डायरेक्टरी का पाथ सेट करें। `dataDir` वेरिएबल में `"Your Document Directory"` को वास्तविक पाथ से बदलें।

```csharp
string dataDir = "Your Document Directory";
```

> **प्रो टिप:** एक एब्सोल्यूट पाथ का उपयोग करने से जब एप्लिकेशन अलग कार्य निर्देशिका से चलता है तो भ्रम समाप्त हो जाता है।

### चरण 2: SevenZip एंट्रीज़ बनाएं (compress directory to 7z)

अब, प्रक्रिया के कोर में डुबकी लगाते हैं – SevenZip एंट्रीज़ बनाना। यह चरण **compress directory to 7z** फ़ॉर्मेट में डायरेक्टरी को संकुचित करता है।

```csharp
//ExStart: CreateSevenZipEntries
using (SevenZipArchive archive = new SevenZipArchive())
{
    archive.CreateEntries(dataDir);
    archive.Save("SevenZip.7z");
}
//ExEnd: CreateSevenZipEntries
```

उपरोक्त कोड एक `SevenZipArchive` को इनिशियलाइज़ करता है, निर्दिष्ट फ़ोल्डर से हर फ़ाइल जोड़ता है, और संकुचित आर्काइव को **SevenZip.7z** में लिखता है।

### चरण 3: सफलता संदेश दिखाएँ

सुनिश्चित करने के लिए कि सब कुछ सुचारू रूप से हुआ, एक सफलता संदेश दिखाएँ:

```csharp
Console.WriteLine("Successfully Created a Seven Zip File");
```

अब आपके पास एक तैयार‑से‑शेयर करने योग्य **7z आर्काइव** है जिसे ट्रांसफ़र, स्टोर या आगे प्रोसेस किया जा सकता है।
