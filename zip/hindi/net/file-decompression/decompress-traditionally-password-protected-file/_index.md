# पासवर्ड के साथ ज़िप निकालें Aspose.Zip for .NET का उपयोग करके

पासवर्ड के साथ ज़िप निकालना .NET डेवलपर्स के लिए एक सामान्य कार्य है जो गोपनीय फ़ाइलों को सुरक्षित या साझा करना चाहते हैं। इस ट्यूटोरियल में आप **कैसे पासवर्ड के साथ ज़िप निकालें** सीखेंगे **Aspose.Zip for .NET** लाइब्रेरी का उपयोग करके, और देखेंगे कि वही तरीका आपको **पासवर्ड‑सुरक्षित ज़िप** आर्काइव को डिकम्प्रेस करने, **पासवर्ड‑सुरक्षित ज़िप** फ़ाइलों को अनज़िप करने, और कुछ ही लाइनों के कोड से **c# unzip password protected** ऑपरेशन करने की अनुमति देता है।

## त्वरित उत्तर
- **ज़िप फ़ाइलों को संभालने के लिए मुख्य क्लास कौन सी है?** `Archive` Aspose.Zip नेमस्पेस से।  
- **कौन सा मेथड पासवर्ड प्रदान करता है?** `ArchiveLoadOptions` के माध्यम से `DecryptionPassword` पास करें।  
- **क्या मैं .NET Core में पासवर्ड‑सुरक्षित ज़िप फ़ाइलें अनज़िप कर सकता हूँ?** हाँ, Aspose.Zip .NET Framework, .NET Core, और .NET 5/6+ को सपोर्ट करता है।  
- **क्या विकास के लिए लाइसेंस चाहिए?** परीक्षण के लिए एक अस्थायी लाइसेंस पर्याप्त है; उत्पादन के लिए पूर्ण लाइसेंस आवश्यक है।  
- **कोड की कितनी लाइनों की जरूरत है?** 20 लाइनों से कम (using स्टेटमेंट्स को छोड़कर)।

## “पासवर्ड के साथ ज़िप निकालें” क्या है?
पासवर्ड के साथ ज़िप निकालना का मतलब है एन्क्रिप्टेड ZIP आर्काइव को पढ़ना, सही पासवर्ड प्रदान करना, और लाइब्रेरी को फ़ाइलों को डिक्रिप्ट और अनपैक करने देना। यही **पासवर्ड‑सुरक्षित ज़िप एक्सट्रैक्शन** का मूल है।

## इस कार्य के लिए Aspose.Zip क्यों उपयोग करें?
- **पूर्ण .NET समर्थन** – .NET Framework, .NET Core, और नवीनतम .NET संस्करणों के साथ काम करता है।  
- **पारंपरिक एन्क्रिप्शन हैंडलिंग** – कई पुराने टूल्स द्वारा उपयोग किए जाने वाले लेगेसी ZipCrypto मेथड को सपोर्ट करता है।  
- **सरल API** – पासवर्ड देने और एंट्री पढ़ने के लिए केवल कुछ कॉल्स की जरूरत होती है।  
- **प्रदर्शन‑ऑप्टिमाइज़्ड** – स्ट्रीम्स को कुशलता से प्रोसेस करता है, जिससे बड़े आर्काइव के लिए उपयुक्त है।  
- **asp zip .net** सक्रिय रूप से मेंटेन किया जाता है और व्यापक दस्तावेज़ीकरण शामिल करता है।

## पूर्वापेक्षाएँ

- NuGet (`Aspose.Zip`) के माध्यम से Aspose.Zip for .NET लाइब्रेरी जोड़ें।  
- C# फ़ाइल I/O की बुनियादी जानकारी।

## नेमस्पेस इम्पोर्ट करें
पहले, आवश्यक नेमस्पेस को स्कोप में लाएँ:

```csharp
using Aspose.Zip;
using System.IO;
```

## चरण 1: पासवर्ड‑सुरक्षित ZIP बनाएं (फ़ाइल पर पासवर्ड प्रोटेक्शन चलाएँ)
एक्सट्रैक्शन दिखाने से पहले हमें एक ऐसा ज़िप चाहिए जो पारंपरिक पासवर्ड से सुरक्षित हो। नीचे दिया गया स्निपेट ऐसा आर्काइव बनाता है (यदि आपके पास पहले से कोई है तो उसे पुनः उपयोग कर सकते हैं):

```csharp
string dataDir = "Your Document Directory";
PasswordProtectArchiveWithTraditionalPassword.Run(); // Run password protection on a file example to use it later
```

> **प्रो टिप:** `"Your Document Directory"` को उस पूर्ण पाथ से बदलें जहाँ आप अपनी टेस्ट फ़ाइलें स्टोर करते हैं।

## चरण 2: पारंपरिक पासवर्ड‑सुरक्षित फ़ाइल को डिकम्प्रेस करें
अब कंटेंट निकालते हैं। नीचे दिया गया कोड दिखाता है कि **c# unzip password protected** आर्काइव को Aspose.Zip का उपयोग करके कैसे निकाला जाए:

```csharp
// ExStart: DecompressTraditionallyPasswordProtectedFile
using (FileStream fs = File.OpenRead(dataDir + "CompressWithTraditionalEncryption_out.zip"))
{
    using (var extracted = File.Create(dataDir + "alice_extracted_out.txt"))
    {
        using (Archive archive = new Archive(fs, new ArchiveLoadOptions() { DecryptionPassword = "p@s$" }))
        {
            using (var decompressed = archive.Entries[0].Open())
            {
                byte[] b = new byte[8192];
                int bytesRead;
                while (0 < (bytesRead = decompressed.Read(b, 0, b.Length)))
                {
                    extracted.Write(b, 0, bytesRead);
                }
            }
        }
    }
}
}
// ExEnd: DecompressTraditionallyPasswordProtectedFile
```

इस स्निपेट में हम:

1. एन्क्रिप्टेड ZIP फ़ाइल को रीड‑ओनली स्ट्रीम के रूप में खोलते हैं।  
2. एक नई फ़ाइल (`alice_extracted_out.txt`) बनाते हैं जहाँ डिकम्प्रेस्ड डेटा लिखा जाएगा।  
3. `ArchiveLoadOptions` के साथ `Archive` को इंस्टैंशिएट करते हैं, पासवर्ड (`"p@s$"`) पास करते हुए।  
4. आर्काइव में पहली एंट्री (मान लेते हैं कि केवल एक फ़ाइल है) तक पहुँचते हैं और उसके बाइट्स को आउटपुट फ़ाइल में कॉपी करते हैं।

जब कोड समाप्त हो जाएगा, तो आप सफलतापूर्वक **पासवर्ड के साथ ज़िप निकालेंगे** और मूल फ़ाइल कंटेंट प्राप्त करेंगे।

## सामान्य समस्याएँ और समाधान
| समस्या | कारण | समाधान |
|--------|------|--------|
| “Invalid password” अपवाद | गलत पासवर्ड स्ट्रिंग या `DecryptionPassword` नहीं दिया गया | पासवर्ड दोबारा जाँचें और सुनिश्चित करें कि वह `ArchiveLoadOptions` के माध्यम से दिया गया है। |
| कोई एंट्री नहीं मिली | आर्काइव खाली है या पाथ गलत है | ZIP फ़ाइल पाथ सत्यापित करें और 7‑Zip जैसे टूल से आर्काइव जांचें। |
| बड़े फ़ाइलों से मेमोरी प्रेशर | पूरी फ़ाइल को मेमोरी में पढ़ना | बफ़रड रीड/राइट लूप (जैसा दिखाया गया) का उपयोग करके डेटा को चंक्स में प्रोसेस करें। |
