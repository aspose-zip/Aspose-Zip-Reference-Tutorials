# फ़ाइलों को tar में जोड़ें और Aspose.Zip for .NET का उपयोग करके TarBz2 में संपीड़ित करें

## परिचय

Aspose.Zip for .NET का उपयोग करके **फ़ाइलों को tar में जोड़ने** और उन्हें TarBz2 फ़ॉर्मेट में संपीड़ित करने के हमारे व्यापक गाइड में आपका स्वागत है। चाहे आप बैकअप यूटिलिटी बना रहे हों, डिप्लॉयमेंट पैकेज वितरित कर रहे हों, या बस वितरण के लिए एक कॉम्पैक्ट आर्काइव की आवश्यकता हो, यह ट्यूटोरियल स्पष्ट व्याख्याओं और वास्तविक‑दुनिया के टिप्स के साथ हर चरण को आपके सामने रखता है।

शुरू करने से पहले, सुनिश्चित करें कि आपके पास सभी आवश्यक चीज़ें हैं।

## त्वरित उत्तर
- **कौन सी लाइब्रेरी उपयोग करनी चाहिए?** Aspose.Zip for .NET
- **इम्प्लीमेंटेशन में कितना समय लगेगा?** लगभग 5‑10 मिनट
- **क्या लाइसेंस की आवश्यकता है?** प्रोडक्शन के लिए एक अस्थायी लाइसेंस आवश्यक है; एक मुफ्त ट्रायल उपलब्ध है
- **क्या मैं कई फ़ाइलें संपीड़ित कर सकता हूँ?** हाँ – आप Tar आर्काइव में जितनी चाहें एंट्रीज़ जोड़ सकते हैं
- **क्या यह .NET 6+ के साथ संगत है?** बिल्कुल, Aspose.Zip .NET Framework और .NET Core/5/6 को सपोर्ट करता है

## “फ़ाइलों को tar में जोड़ना” क्या है?
फ़ाइलों को **tar** (Tape Archive) में जोड़ने से एक एकल अनकम्प्रेस्ड कंटेनर बनता है जो डायरेक्टरी स्ट्रक्चर और फ़ाइल मेटाडेटा को संरक्षित करता है। जब आप इसके बाद Bzip2 संपीड़न लागू करते हैं, तो परिणाम **tar.bz2** (TarBz2) आर्काइव होता है—जो कुशल स्टोरेज और ट्रांसफ़र के लिए आदर्श है।

## Aspose.Zip के साथ फ़ाइलों को TarBz2 में संपीड़ित क्यों करें?
- **गति और सरलता** – एक‑लाइन API कॉल्स दोनों tar निर्माण और Bzip2 संपीड़न को संभालते हैं।
- **क्रॉस‑प्लेटफ़ॉर्म** – Windows, Linux, और macOS .NET रनटाइम्स पर काम करता है।
- **सूक्ष्म नियंत्रण** – आप कौन सी फ़ाइलें शामिल करेंगे, कस्टम एंट्री नाम सेट करेंगे, और सीधे डिस्क पर स्ट्रीम करेंगे।

## नेमस्पेस आयात करें

पहले, आवश्यक नेमस्पेस आयात करें ताकि आप Aspose.Zip के Tar और Bzip2 क्लासेज़ तक पहुँच सकें।

```csharp
using System;
using System.ComponentModel;
using Aspose.Zip.Bzip2;
using Aspose.Zip.Tar;
```

## चरण 1: डॉक्यूमेंट डायरेक्टरी सेट करें

उस पाथ को परिभाषित करें जो उन फ़ाइलों वाले फ़ोल्डर की ओर इशारा करता है जिन्हें आप आर्काइव करना चाहते हैं।

```csharp
string dataDir = "Your Document Directory";
```

> `"Your Document Directory"` को अपने स्रोत फ़ोल्डर के पूर्ण या सापेक्ष पाथ से बदलें।

## चरण 2: फ़ाइलों को tar में जोड़ें और TarBz2 आर्काइव बनाएं

प्रक्रिया का मूल भाग `TarArchive` बनाना, एंट्रीज़ जोड़ना, और फिर उसे `Bzip2Archive` से रैप करना है। नीचे दिया गया कोड **tarbz2** बनाने को एक साफ़, डिस्पोज़ेबल‑पैटर्न शैली में दर्शाता है।

```csharp
//ExStart: CompressFile
using (Bzip2Archive bz2 = new Bzip2Archive())
{
    using (TarArchive archive = new TarArchive())
    {
        archive.CreateEntry("alice29.txt", dataDir + "alice29.txt");
        archive.CreateEntry("lcet10.txt", dataDir + "lcet10.txt");

        bz2.SetSource(archive);
        bz2.Save(dataDir + "archive.tar.bz2");
    }
}
```

- `CreateEntry` प्रत्येक फ़ाइल को **tar** कंटेनर में जोड़ता है।  
- `bz2.SetSource(archive)` Bzip2 आर्काइव को पूरे tar स्ट्रीम को संपीड़ित करने के लिए बताता है।  
- `bz2.Save(...)` अंतिम **tar.bz2** फ़ाइल को डिस्क पर लिखता है।

**टिप:** कई फ़ाइलों को **tarbz2** में संपीड़ित करने के लिए, बस प्रत्येक फ़ाइल के लिए `archive.CreateEntry` दोहराएँ।
