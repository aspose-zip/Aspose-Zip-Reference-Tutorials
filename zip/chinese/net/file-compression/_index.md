## 常见问题

**Q: 我可以在 .NET Core 上使用 Aspose.Zip 创建 zip archive .net 吗？**  
A: 当然可以。Aspose.Zip 完全支持 .NET Core 3.1 及更高版本，以及 .NET 5/6/7。

**Q: 如何在不先解压的情况下修改现有归档？**  
A: 使用 “modify zip file c#” API——您可以直接在归档流上添加、删除或重命名条目。

**Q: 哪种压缩方法对文本文件的压缩率最高？**  
A: 通常，Bzip2 或 LZMA 对文本提供更高的压缩率，但速度慢于 Deflate。请两者都测试，以找到合适的平衡。

**Q: 在创建 zip 归档时是否有办法加密文件？**  
A: 是的，Aspose.Zip 支持 AES 加密。您可以在保存前为 ZipArchive 对象设置密码，从而实现 **zip file encryption c#**。

**Q: 我需要担心文件名编码吗？**  
A: Aspose.Zip 会自动处理 Unicode 文件名，但如果需要与旧系统兼容，您也可以显式设置编码。
