## 常见问题

**问：我可以在 .NET Core 上使用 Aspose.Zip 创建 zip archive .net 吗？**  
答：当然可以。Aspose.Zip 完全支持 .NET Core 3.1 及更高版本，以及 .NET 5/6/7。

**问：如何在不先解压的情况下修改已有的存档？**  
答：使用 “modify zip file c#” API——您可以直接在存档流上添加、删除或重命名条目。

**问：哪种压缩方法对文本文件的压缩率最高？**  
答：通常，Bzip2 或 LZMA 对文本提供更高的压缩率，但它们比 Deflate 更慢。请两者都测试以找到合适的平衡。

**问：在创建 zip 存档时是否可以加密文件？**  
答：可以，Aspose.Zip 支持 AES 加密。您可以在保存之前为 ZipArchive 对象设置密码。

**问：我需要担心文件名编码吗？**  
答：Aspose.Zip 会自动处理 Unicode 文件名，但如果需要兼容旧系统，您可以显式设置编码。
