## 常见问题

**Q: 我可以在同一个应用程序中同时使用 Aspose.Zip 进行压缩和解压归档吗？**  
A: 可以，API 提供创建、更新和提取 ZIP 文件的方法，帮助您完整管理归档生命周期。

**Q: 如何控制特定文件的压缩级别？**  
A: 在将文件添加到归档之前，设置 `ZipEntry` 上的 `CompressionLevel` 属性。更高的级别会生成更小的文件，但处理时间更长。

**Q: 能否在不重新压缩整个归档的情况下向已有 ZIP 添加文件？**  
A: 完全可以。使用 `Update` 模式追加新条目，既保留已有数据，又加快操作速度。

**Q: Aspose.Zip 是否支持密码保护的归档？**  
A: 支持，您可以在创建或提取归档时指定密码，以保护内容安全。

**Q: Aspose.Zip 兼容哪些 .NET 版本？**  
A: 该库兼容 .NET Framework 4.5+、.NET Core 3.1+、.NET 5、.NET 6 以及更高版本。
