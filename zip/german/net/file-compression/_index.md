## Häufig gestellte Fragen

**Q: Kann ich Aspose.Zip sowohl zum Komprimieren als auch zum Extrahieren von Archiven in derselben Anwendung verwenden?**  
A: Ja, die API bietet Methoden zum Erstellen, Aktualisieren und Extrahieren von ZIP‑Dateien, sodass Sie den gesamten Archivlebenszyklus verwalten können.

**Q: Wie steuere ich die Komprimierungsstufe für eine bestimmte Datei?**  
A: Setzen Sie die `CompressionLevel`‑Eigenschaft auf dem `ZipEntry`, bevor Sie die Datei zum Archiv hinzufügen. Höhere Stufen ergeben kleinere Dateien, benötigen jedoch mehr Verarbeitungszeit.

**Q: Ist es möglich, Dateien zu einem bestehenden ZIP hinzuzufügen, ohne das gesamte Archiv neu zu komprimieren?**  
A: Absolut. Verwenden Sie den `Update`‑Modus, um neue Einträge anzuhängen, wodurch vorhandene Daten erhalten bleiben und der Vorgang beschleunigt wird.

**Q: Unterstützt Aspose.Zip passwortgeschützte Archive?**  
A: Ja, Sie können beim Erstellen oder Extrahieren eines Archivs ein Passwort angeben, um dessen Inhalt zu sichern.

**Q: Welche .NET‑Versionen sind mit Aspose.Zip kompatibel?**  
A: Die Bibliothek funktioniert mit .NET Framework 4.5+, .NET Core 3.1+, .NET 5, .NET 6 und späteren Versionen.
