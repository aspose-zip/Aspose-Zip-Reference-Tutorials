# Πώς να δημιουργήσετε αρχεία 7z με κρυπτογράφηση AES στο .NET

## Εισαγωγή

Η δημιουργία ενός **7z** αρχείου με ισχυρή κρυπτογράφηση AES είναι μια συχνή απαίτηση όταν χρειάζεται να προστατεύσετε ευαίσθητα δεδομένα στις .NET εφαρμογές σας. Σε αυτό το σεμινάριο θα σας δείξουμε **πώς να δημιουργήσετε αρχεία 7z** με ασφάλεια χρησιμοποιώντας τη βιβλιοθήκη Aspose.Zip, καλύπτοντας τα πάντα από τη ρύθμιση του έργου μέχρι την προσθήκη κρυπτογραφημένων καταχωρίσεων. Στο τέλος, θα καταλάβετε γιατί η **ασφαλής αρχειοθέτηση .NET** είναι σημαντική, πώς λειτουργεί η **aes zip encryption** και πώς να εφαρμόσετε **προστασία κωδικού zip** με λίγες γραμμές κώδικα C#.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “7z”;** Είναι η επέκταση αρχείου για αρχεία που δημιουργούνται με τη μορφή 7‑Zip, η οποία υποστηρίζει υψηλή αναλογία συμπίεσης και ισχυρή κρυπτογράφηση.  
- **Ποιος αλγόριθμος παρέχει την καλύτερη ασφάλεια;** AES‑256, διαθέσιμο μέσω του `SevenZipAESEncryptionSettings` του Aspose.Zip.  
- **Χρειάζομαι άδεια για ανάπτυξη;** Διατίθεται προσωρινή άδεια για δοκιμές· απαιτείται πλήρης άδεια για παραγωγή.  
- **Μπορώ να κρυπτογραφήσω πολλαπλές καταχωρίσεις;** Ναι—επαναλάβετε την κλήση `CreateEntry` για κάθε αρχείο που θέλετε να προστατέψετε.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, και .NET 5/6+.

## Τι είναι ένα αρχείο 7z και γιατί να το κρυπτογραφήσετε;

Ένα αρχείο **7z** είναι ένα συμπιεσμένο κοντέινερ που μπορεί να περιέχει πολλά αρχεία και καταλόγους. Επειδή υποστηρίζει κρυπτογράφηση AES, είναι ιδανικό για σενάρια όπου η εμπιστευτικότητα των δεδομένων είναι κρίσιμη—όπως η μετάδοση εμπιστευτικών αναφορών, η δημιουργία αντιγράφων ασφαλείας κωδικοποιημένου κώδικα ή η αποθήκευση προσωπικών πληροφοριών. Η χρήση **κρυπτογραφημένων seven zip** αρχείων σας βοηθά να τηρήσετε τις απαιτήσεις συμμόρφωσης και προστατεύει τα δεδομένα από μη εξουσιοδοτημένη πρόσβαση.

## Βήμα 2: Δημιουργία Seven Zip Αρχείου με Κρυπτογράφηση AES

Τώρα θα δημιουργήσουμε το **7z** αρχείο και θα προσθέσουμε μια κρυπτογραφημένη καταχώριση. Το παράδειγμα χρησιμοποιεί έναν απλό πίνακα byte, αλλά μπορείτε να το αντικαταστήσετε με οποιοδήποτε ρεύμα (π.χ., ένα ρεύμα αρχείου).

```csharp
//ExStart: ArchiveWithEncryptedEntry
using (FileStream sevenZipFile = File.Open("archive.7z", FileMode.Create))
{
    using (var archive = new SevenZipArchive())
    {
        archive.CreateEntry("entry1.bin", 
            new MemoryStream(new byte[] { 0x00, 0xFF }), 
            new SevenZipEntrySettings(new SevenZipLZMACompressionSettings(), new SevenZipAESEncryptionSettings("test1")));
        archive.Save(sevenZipFile);
    }
}
//ExEnd: ArchiveWithEncryptedEntry
Console.WriteLine("Successfully Created a Seven Zip File with AES Encryption Settings");
```

**Επεξήγηση:**  
- `SevenZipArchive` δημιουργεί ένα νέο 7z κοντέινερ.  
- `CreateEntry` προσθέτει ένα αρχείο με όνομα `entry1.bin`.  
- `SevenZipAESEncryptionSettings("test1")` εφαρμόζει **aes zip encryption** με τον κωδικό *test1*.  
- Μπορείτε να επαναλάβετε το `CreateEntry` για επιπλέον αρχεία, το καθένα με τις δικές του ρυθμίσεις κρυπτογράφησης εάν χρειάζεται.
