## Βήμα 2: Δημιουργία Seven Zip Αρχείου με AES Κρυπτογράφηση

```csharp
//ExStart: ArchiveWithEncryptedEntry
using (FileStream sevenZipFile = File.Open("archive.7z", FileMode.Create))
{
    using (var archive = new SevenZipArchive())
    {
        archive.CreateEntry("entry1.bin", 
            new MemoryStream(new byte[] { 0x00, 0xFF }), 
            new SevenZipEntrySettings(new SevenZipLZMACompressionSettings(), new SevenZipAESEncryptionSettings("test1")));
        archive.Save(sevenZipFile);
    }
}
//ExEnd: ArchiveWithEncryptedEntry
Console.WriteLine("Successfully Created a Seven Zip File with AES Encryption Settings");
```

**Εξήγηση:**  
Σε αυτό το βήμα ανοίγουμε (ή δημιουργούμε) ένα αρχείο με όνομα **archive.7z**, προσθέτουμε μια μοναδική καταχώρηση με όνομα **entry1.bin**, και εφαρμόζουμε **AES κρυπτογράφηση** με τον κωδικό `"test1"`. Το αντικείμενο `SevenZipLZMACompressionSettings` ελέγχει τον αλγόριθμο συμπίεσης, ενώ το `SevenZipAESEncryptionSettings` διαχειρίζεται την κρυπτογράφηση. Μπορείτε να επαναλάβετε την κλήση `CreateEntry` για να προσθέσετε περισσότερα αρχεία, το καθένα με τις δικές του ρυθμίσεις κρυπτογράφησης αν χρειάζεται.

### Πώς να κρυπτογραφήσετε ξεχωριστά zip entry
Αν χρειάζεται να **κρυπτογραφήσετε zip entry** αντικείμενα με διαφορετικούς κωδικούς, απλώς δημιουργήστε ένα νέο `SevenZipAESEncryptionSettings` για κάθε κλήση στο `CreateEntry`.

### Πώς να προστατεύσετε 7z αρχεία με κωδικό πρόσβασης
Ο κωδικός που παρέχετε στο `SevenZipAESEncryptionSettings` λειτουργεί ως **προστασία κωδικού πρόσβασης** για ολόκληρο το αρχείο. Όταν ανοίγετε το αρχείο, πρέπει να δοθεί ο ίδιος κωδικός.
