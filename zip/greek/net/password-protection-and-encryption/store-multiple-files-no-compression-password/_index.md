# Aspose.Zip για .NET - Εκμάθηση Προστασίας Αρχείου Zip με Κωδικό

Στη σύγχρονη ανάπτυξη .NET, η ασφαλής αρχειοθέτηση αρχείων είναι συχνή απαίτηση. Με **Aspose.Zip για .NET**, μπορείτε να **προστατεύετε με κωδικό αρχεία zip**, να αποθηκεύετε πολλά στοιχεία χωρίς καμία συμπίεση και να εφαρμόζετε ισχυρή κρυπτογράφηση AES—όλα με λίγες γραμμές C#. Αυτό το tutorial σας καθοδηγεί βήμα‑βήμα για τη δημιουργία zip που περιέχει πολλαπλά αρχεία, χρησιμοποιεί λειτουργία *store* (χωρίς συμπίεση) και είναι κλειδωμένο με κωδικό.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “προστασία zip αρχείου με κωδικό”;** Κρυπτογραφεί το περιεχόμενο του zip ώστε να μπορεί να ανοιχθεί μόνο με τον σωστό κωδικό.  
- **Ποιος αλγόριθμος κρυπτογράφησης χρησιμοποιείται;** AES‑256 μέσω `AesEcryptionSettings`.  
- **Μπορώ να προσθέσω περισσότερα από ένα αρχεία;** Ναι – επαναλάβετε την κλήση `CreateEntry` για κάθε αρχείο προέλευσης.  
- **Χρειάζεται άδεια για παραγωγή;** Απαιτείται εμπορική άδεια· διατίθεται δωρεάν δοκιμαστική έκδοση.  
- **Υποστηρίζεται στο .NET 6/7;** Απόλυτα – το Aspose.Zip λειτουργεί με .NET Framework, .NET Core και .NET 5/6/7.

## Τι είναι η προστασία zip αρχείου με κωδικό;
Μια *προστασία zip αρχείου με κωδικό* είναι ένα αρχείο ZIP του οποίου οι καταχωρήσεις κρυπτογραφούνται με κωδικό που ορίζει ο χρήστης. Όταν ανοίγεται το αρχείο, πρέπει να δοθεί ο κωδικός· διαφορετικά το περιεχόμενο παραμένει μη αναγνώσιμο. Το Aspose.Zip υλοποιεί αυτό μέσω κρυπτογράφησης AES‑256, προσφέροντας ισχυρή ασφάλεια για ευαίσθητα δεδομένα.

## Γιατί να χρησιμοποιήσετε προστασία zip με κωδικό στο Aspose.Zip;
- **Αποθήκευση χωρίς συμπίεση** – `StoreCompressionSettings` διατηρεί το αρχικό μέγεθος του αρχείου, ιδανικό για ήδη συμπιεσμένα μέσα.  
- **Ισχυρή κρυπτογράφηση** – AES‑256 προστατεύει τα δεδομένα από επιθέσεις brute‑force.  
- **Πλήρη ενσωμάτωση .NET** – Λειτουργεί με .NET Framework, .NET Core και .NET 5/6/7 χωρίς εξαρτήσεις σε native κώδικα.  
- **Απλό API** – Δημιουργήστε το αρχείο, ορίστε κωδικό, προσθέστε καταχωρήσεις και αποθηκεύστε – όλα σε λίγες γραμμές.

## Προαπαιτούμενα

Πριν βουτήξουμε στον κώδικα, βεβαιωθείτε ότι έχετε:

- **Aspose.Zip για .NET** εγκατεστημένο. Μπορείτε να το κατεβάσετε [εδώ](https://releases.aspose.com/zip/net/).  
- Έναν φάκελο που περιέχει τα αρχεία που θέλετε να αρχειοθετήσετε. Στα παραδείγματα παρακάτω, η μεταβλητή `dataDir` δείχνει σε αυτόν το φάκελο.

## Εισαγωγή Namespaces

Πρώτα, φέρετε τα απαιτούμενα namespaces στο scope:

```csharp
// The path to the resource directory.
string dataDir = "Your Document Directory"

// Import Aspose.Zip namespaces
using Aspose.Zip;
using Aspose.Zip.Settings.Compression;
using Aspose.Zip.Settings.Encryption;
```

## Πώς να προστατεύσετε zip αρχείο με κωδικό και να αποθηκεύσετε πολλαπλά αρχεία χωρίς συμπίεση

Ακολουθεί ο οδηγός βήμα‑βήμα. Κάθε βήμα περιλαμβάνει σύντομη εξήγηση και τον αρχικό κώδικα (αμετάβλητο).

### Βήμα 1: Άνοιγμα του Zip Αρχείου

Δημιουργούμε ένα νέο `FileStream` που θα κρατήσει το τελικό αρχείο.

```csharp
//ExStart: StoreMutlipleFilesWithoutCompressionWithPassword
using (FileStream zipFile = File.Open(dataDir + "StoreMutlipleFilesWithoutCompressionWithPassword_out.zip", FileMode.Create))
{
```

### Βήμα 2: Άνοιγμα του Πηγαίου Αρχείου

Ανοίξτε το πρώτο αρχείο που θέλετε να προσθέσετε στο zip. Μπορείτε να επαναλάβετε αυτό το μπλοκ για επιπλέον αρχεία.

```csharp
using (FileStream source1 = File.Open(dataDir + "alice29.txt", FileMode.Open, FileAccess.Read))
{
```

### Βήμα 3: Δημιουργία Αρχείου με Αποθήκευση χωρίς Συμπίεση και Κρυπτογράφηση AES

Εδώ ρυθμίζουμε το αρχείο ώστε **να αποθηκεύει** (χωρίς συμπίεση) τα αρχεία και να εφαρμόζει **προστασία zip αρχείου με κωδικό** χρησιμοποιώντας AES‑256.

```csharp
using (var archive = new Archive(new ArchiveEntrySettings(new StoreCompressionSettings(), new AesEcryptionSettings("p@s$", EncryptionMethod.AES256))))
{
```

### Βήμα 4: Δημιουργία Καταχώρησης Αρχείου και Αποθήκευση – *create archive entry c#*

Τώρα προσθέτουμε το αρχείο στο zip και γράφουμε το κρυπτογραφημένο αρχείο στο δίσκο.

```csharp
archive.CreateEntry("alice29.txt", source1);
archive.Save(zipFile);
```

> **Συμβουλή:** Για να προσθέσετε περισσότερα αρχεία, απλώς καλέστε `archive.CreateEntry("anotherFile.txt", anotherStream);` πριν από `archive.Save(zipFile);`.

## Συχνά Προβλήματα και Λύσεις

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| **Εξαίρεση “Invalid password”** | Λάθος κωδικός ή μη συμβατική μέθοδος κρυπτογράφησης. | Βεβαιωθείτε ότι η συμβολοσειρά κωδικού στο `AesEcryptionSettings` ταιριάζει με αυτόν που θα χρησιμοποιήσετε για το άνοιγμα του zip, και ότι χρησιμοποιείτε `EncryptionMethod.AES256`. |
| **Μέγεθος αρχείου μεγαλύτερο από το αναμενόμενο** | Χρήση συμπίεσης κατά λάθος. | Επαληθεύστε ότι χρησιμοποιείτε `StoreCompressionSettings` (χωρίς συμπίεση) αντί για `DeflateCompressionSettings`. |
| **Το stream δεν κλείνει** | Λείπει η αγκύλη κλεισίματος για τις δηλώσεις `using`. | Σιγουρευτείτε ότι κάθε μπλοκ `using` κλείνει σωστά· το δείγμα κώδικα δείχνει τη σωστή εσοχή. |
