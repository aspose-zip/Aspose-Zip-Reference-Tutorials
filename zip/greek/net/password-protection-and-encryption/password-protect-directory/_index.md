## Βήμα 2: Προστασία Καταλόγου με Κωδικό
Χρησιμοποιήστε το `TraditionalEncryptionSettings` για να ορίσετε τον κωδικό και να δημιουργήσετε το κρυπτογραφημένο αρχείο. Αυτό αποτελεί τον πυρήνα της **c# zip password protection**.

```csharp
//ExStart: PasswordProtectDirectory
using (FileStream zipFile = File.Open(dataDir + "all_corpus_encrypted_out.zip", FileMode.Create))
{
    DirectoryInfo corpus = new DirectoryInfo(".\\CanterburyCorpus");
    using (var archive = new Archive(new ArchiveEntrySettings(null, new TraditionalEncryptionSettings("p@s$"))))
    {
        archive.CreateEntries(corpus);
        archive.Save(zipFile);
        //ExEnd: PasswordProtectDirectory
    }
}
```

## Βήμα 3: Επεξήγηση του Κώδικα
- **Δημιουργία του αρχείου εξόδου:** `File.Open(..., FileMode.Create)` ανοίγει (ή δημιουργεί) το αρχείο ZIP που θα περιέχει τα κρυπτογραφημένα δεδομένα.  
- **Επιλογή του φακέλου προέλευσης:** `new DirectoryInfo(".\\CanterburyCorpus")` ενημερώνει το Aspose.Zip ποιον κατάλογο θα συμπιέσει.  
- **Εφαρμογή του κωδικού:** `new TraditionalEncryptionSettings("p@s$")` ορίζει τον κωδικό που θα προστατεύει το αρχείο.  
- **Προσθήκη καταχωρήσεων & αποθήκευση:** `archive.CreateEntries(corpus)` προσθέτει κάθε αρχείο του φακέλου, και `archive.Save(zipFile)` γράφει το κρυπτογραφημένο ZIP στον δίσκο.
