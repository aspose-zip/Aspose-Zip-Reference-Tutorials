## How to Extract ZIP File Password (Unzip Password Protected ZIP)

### Step 1: Set Up Your Project

Δημιουργήστε ένα νέο έργο C# console ή desktop στο Visual Studio. Προσθέστε μια αναφορά στο Aspose.Zip DLL που κατεβάσατε και αντιγράψτε το κρυπτογραφημένο ZIP αρχείο στον φάκελο εξόδου του έργου (π.χ., `bin\Debug\net6.0`).

### Step 2: Initialize Variables

Ορίστε τον φάκελο που περιέχει τα αρχεία δοκιμής σας. Αυτό διατηρεί τον κώδικα καθαρό και επαναχρησιμοποιήσιμο:

```csharp
string dataDir = "YourDocumentDirectory";
```

### Step 3: Decompress AES Encrypted File

Τώρα φτάνουμε στην κεντρική λογική που πραγματικά **εξάγει τον κωδικό πρόσβασης του zip αρχείου** και διαβάζει την κρυπτογραφημένη καταχώρηση. Το παρακάτω απόσπασμα ανοίγει το αρχείο, παρέχει τον κωδικό (`p@s$` σε αυτό το παράδειγμα) και γράφει το εξαγόμενο περιεχόμενο σε νέο αρχείο.

```csharp
//ExStart: DecompressAESEncryptedFile
using (FileStream fs = File.OpenRead(dataDir + "PasswordProtectWithAES256_out.zip"))
{
    using (var extracted = File.Create(dataDir + "alice_aesextracted_out.txt"))
    {
        using (Archive archive = new Archive(fs))
        {
            using (var decompressed = archive.Entries[0].Open("p@s$"))
            {
                byte[] b = new byte[8192];
                int bytesRead;
                while (0 < (bytesRead = decompressed.Read(b, 0, b.Length)))
                {
                    extracted.Write(b, 0, bytesRead);
                }
            }
        }
    }
}
//ExEnd: DecompressAESEncryptedFile
```

> **Pro tip:** Αν χρειάζεται να εξάγετε πολλαπλές καταχωρήσεις, κάντε βρόχο στο `archive.Entries` και καλέστε `Open(password)` για κάθε μία.

## How to Decompress AES ZIP Files

Η ίδια κλάση `Archive` λειτουργεί για οποιοδήποτε AES‑κρυπτογραφημένο αρχείο, ανεξαρτήτως μήκους κλειδιού (128, 192 ή 256 bits). Απλώς δώστε τον σωστό κωδικό και η βιβλιοθήκη διαχειρίζεται την αποκρυπτογράφηση εσωτερικά.

## Common Issues & Troubleshooting

| Συμπτωμα | Πιθανή Αιτία | Διόρθωση |
|---------|--------------|----------|
| **Εξαίρεση “Invalid password”** | Λάθος κωδικός ή κατεστραμμένο αρχείο | Επαληθεύστε τη συμβολοσειρά κωδικού, βεβαιωθείτε ότι ταιριάζει ακριβώς σε πεζά/κεφαλαία και ειδικούς χαρακτήρες. |
