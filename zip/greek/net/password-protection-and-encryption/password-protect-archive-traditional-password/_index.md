# Δημιουργία ZIP με προστασία κωδικού πρόσβασης με Aspose.Zip για .NET

Στον χώρο της ανάπτυξης .NET, η εκμάθηση του πώς να **δημιουργήσετε zip με προστασία κωδικού** είναι ένα κρίσιμο στοιχείο του σχεδιασμού εφαρμογών. Το Aspose.Zip για .NET προσφέρει μια ισχυρή λύση για την προσθήκη κωδικού σε αρχεία zip χρησιμοποιώντας παραδοσιακή κρυπτογράφηση κωδικού. Αυτός ο οδηγός βήμα‑βήμα θα σας καθοδηγήσει στη διαδικασία, εξασφαλίζοντας ότι τα αρχειοθετημένα δεδομένα σας παραμένουν εμπιστευτικά και ασφαλή.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “create password protected zip”;** Σημαίνει τη δημιουργία ενός αρχείου ZIP του οποίου τα περιεχόμενα είναι κρυπτογραφημένα και μπορούν να ανοιχτούν μόνο με τον σωστό κωδικό πρόσβασης.  
- **Ποια βιβλιοθήκη μπορώ να χρησιμοποιήσω;** Το Aspose.Zip για .NET παρέχει ενσωματωμένη υποστήριξη για παραδοσιακή προστασία κωδικού.  
- **Χρειάζομαι άδεια;** Διατίθεται δωρεάν δοκιμή, αλλά απαιτείται εμπορική άδεια για χρήση σε παραγωγή.  
- **Μπορώ να το χρησιμοποιήσω με .NET Core;** Ναι, η βιβλιοθήκη λειτουργεί με .NET Framework, .NET Core και .NET 5/6+.  
- **Πόσο χρόνο διαρκεί η υλοποίηση;** Συνήθως λιγότερο από 10 λεπτά για ένα βασικό αρχείο με προστασία κωδικού.

## Τι είναι το “create password protected zip”;
Η δημιουργία ενός zip με προστασία κωδικού σημαίνει τη συμπίεση ενός ή περισσότερων αρχείων σε ένα κοντέινερ ZIP και την κρυπτογράφηση του κοντέινερ με έναν κωδικό. Το προκύπτον αρχείο μπορεί να μοιραστεί ή να αποθηκευτεί με ασφάλεια, επειδή τα περιεχόμενά του είναι αδιάβαστα χωρίς τον σωστό κωδικό.

## Γιατί να χρησιμοποιήσετε το Aspose.Zip για προστασία κωδικού αρχείων ZIP;
- **Πλήρης ενσωμάτωση .NET** – λειτουργεί άψογα με έργα C# και VB.NET.  
- **Παραδοσιακή κρυπτογράφηση** – συμβατή με τις περισσότερες εφαρμογές ZIP που υποστηρίζουν προστασία κωδικού.  
- **Χωρίς εξωτερικές εξαρτήσεις** – η βιβλιοθήκη διαχειρίζεται τη συμπίεση, την κρυπτογράφηση και την αποθήκευση με ένα μόνο API.  
- **Βελτιστοποιημένη απόδοση** – κατάλληλη για μικρά αρχεία όπως αρχεία καταγραφής καθώς και για μεγάλα πακέτα εγγράφων.

## Προαπαιτούμενα
Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε:

2. Έναν φάκελο που περιέχει τα αρχεία που θέλετε να συμπιέσετε και να προστατέψετε.

## Εισαγωγή Χώρων Ονομάτων
Πρώτα, εισάγετε τους χώρους ονομάτων που σας δίνουν πρόσβαση στις κλάσεις συμπίεσης και κρυπτογράφησης.

```csharp
using Aspose.Zip;
using System.IO;
using Aspose.Zip.Saving;
```

## Βήμα 1: Άνοιγμα του Καταλόγου Πόρων
Αναγνωρίστε τον κατάλογο που περιέχει τα αρχεία που θέλετε να αρχειοθετήσετε. Αυτή η διαδρομή θα χρησιμοποιηθεί κατά τη δημιουργία του ροής ZIP.

## Βήμα 2: Δημιουργία Αρχείου με Παραδοσιακό Κωδικό
Τώρα θα δημιουργήσουμε το αρχείο και **προσθέσουμε κωδικό στο zip** χρησιμοποιώντας το `TraditionalEncryptionSettings`. Ο κωδικός `"p@s$"` είναι μόνο ένα παράδειγμα· αντικαταστήστε τον με ένα ισχυρό μυστικό της επιλογής σας.

```csharp
//ExStart: PasswordProtectArchiveWithTraditionalPassword
using (FileStream zipFile = File.Open(dataDir + "CompressWithTraditionalEncryption_out.zip", FileMode.Create))
{
    using (FileStream source1 = File.Open(dataDir + "alice29.txt", FileMode.Open, FileAccess.Read))
    {
        var archive = new Archive(new ArchiveEntrySettings(null, new TraditionalEncryptionSettings("p@s$")));
        archive.CreateEntry("alice29.txt", source1);
        archive.Save(zipFile);
    }
}
//ExEnd: PasswordProtectArchiveWithTraditionalPassword 
```

> **Συμβουλή:** Αποθηκεύστε τον κωδικό με ασφάλεια (π.χ., στο Azure Key Vault) αντί να τον κωδικοποιήσετε σκληρά.

## Βήμα 3: Αποθήκευση του Αρχείου
Η κλήση `archive.Save(zipFile);` γράφει την ενέργεια **αποθήκευσης zip με κωδικό** στο δίσκο. Μετά από αυτό το βήμα, το αρχείο `CompressWithTraditionalEncryption_out.zip` είναι ένα πλήρως προστατευμένο με κωδικό αρχείο ZIP έτοιμο για διανομή.
