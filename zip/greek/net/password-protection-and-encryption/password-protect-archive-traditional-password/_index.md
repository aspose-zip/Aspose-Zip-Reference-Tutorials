# Δημιουργία ZIP με Προστασία Κωδικού Πρόσβασης με Aspose.Zip για .NET

Στον χώρο της ανάπτυξης .NET, η εκμάθηση του πώς να **δημιουργήσετε zip με προστασία κωδικού πρόσβασης** είναι ένα κρίσιμο στοιχείο του σχεδιασμού εφαρμογών. Το Aspose.Zip for .NET προσφέρει μια ισχυρή λύση για **προσθήκη κωδικού πρόσβασης σε zip** αρχεία χρησιμοποιώντας παραδοσιακή κρυπτογράφηση κωδικού. Αυτός ο οδηγός βήμα‑βήμα θα σας καθοδηγήσει στη διαδικασία, εξασφαλίζοντας ότι τα αρχειοθετημένα δεδομένα σας παραμένουν εμπιστευτικά και ασφαλή.

- **Ποια βιβλιοθήκη μπορώ να χρησιμοποιήσω;** Το Aspose.Zip for .NET παρέχει ενσωματωμένη υποστήριξη για παραδοσιακή προστασία κωδικού.  
- **Χρειάζομαι άδεια;** Διατίθεται δωρεάν δοκιμή, αλλά απαιτείται εμπορική άδεια για χρήση σε παραγωγή.

## Τι είναι το “create password protected zip”;
Η δημιουργία ενός zip με προστασία κωδικού σημαίνει τη συμπίεση ενός ή περισσότερων αρχείων σε ένα κοντέινερ ZIP και την κρυπτογράφηση του κοντέινερ με έναν κωδικό πρόσβασης. Το προκύπτον αρχείο μπορεί να μοιραστεί ή να αποθηκευτεί με ασφάλεια, επειδή τα περιεχόμενα του είναι αδιάβαστα χωρίς τον σωστό κωδικό.

- **Παραδοσιακή κρυπτογράφηση** – συμβατή με τις περισσότερες εφαρμογές ZIP που υποστηρίζουν προστασία κωδικού.  
- **Χωρίς εξωτερικές εξαρτήσεις** – η βιβλιοθήκη διαχειρίζεται τη συμπίεση, την κρυπτογράφηση και την αποθήκευση σε ένα μόνο API.  
- **Βελτιστοποιημένη απόδοση** – κατάλληλη για μικρά αρχεία όπως αρχεία καταγραφής, καθώς και για μεγάλα πακέτα εγγράφων.

## Εισαγωγή Namespaces
Πρώτα, εισάγετε τα namespaces που σας δίνουν πρόσβαση στις κλάσεις συμπίεσης και κρυπτογράφησης.

using Aspose.Zip;

## Βήμα 1: Άνοιγμα του Καταλόγου Πόρων
Εντοπίστε τον κατάλογο που περιέχει τα αρχεία που θέλετε να αρχειοθετήσετε. Αυτή η διαδρομή θα χρησιμοποιηθεί κατά τη δημιουργία του ροής ZIP.

## Βήμα 2: Δημιουργία Αρχείου με Παραδοσιακό Κωδικό
Τώρα θα δημιουργήσουμε το αρχείο και θα **προσθέσουμε κωδικό σε zip** χρησιμοποιώντας το `TraditionalEncryptionSettings`. Ο κωδικός `"p@s$"` είναι μόνο ένα παράδειγμα· αντικαταστήστε τον με ένα ισχυρό μυστικό της επιλογής σας.

```csharp
//ExStart: PasswordProtectArchiveWithTraditionalPassword
using (FileStream zipFile = File.Open(dataDir + "CompressWithTraditionalEncryption_out.zip", FileMode.Create))
{
    using (FileStream source1 = File.Open(dataDir + "alice29.txt", FileMode.Open, FileAccess.Read))
    {
        var archive = new Archive(new ArchiveEntrySettings(null, new TraditionalEncryptionSettings("p@s$")));
        archive.CreateEntry("alice29.txt", source1);
        archive.Save(zipFile);
    }
}
//ExEnd: PasswordProtectArchiveWithTraditionalPassword 
```

> **Συμβουλή:** Αποθηκεύστε τον κωδικό με ασφάλεια (π.χ., στο Azure Key Vault) αντί να τον κωδικοποιήσετε σκληρά στο κώδικα.

## Βήμα 3: Αποθήκευση του Αρχείου
Η κλήση `archive.Save(zipFile);` γράφει τη λειτουργία **αποθήκευσης zip με κωδικό** στο δίσκο. Μετά από αυτό το βήμα, το αρχείο `CompressWithTraditionalEncryption_out.zip` είναι ένα πλήρως προστατευμένο με κωδικό ZIP αρχείο έτοιμο για διανομή.

## Πώς να προσθέσετε κωδικό σε zip αρχεία με Aspose.Zip .NET
Όταν καλείτε το `TraditionalEncryptionSettings`, λέτε στο Aspose.Zip να χρησιμοποιήσει τον παλαιό αλγόριθμο κρυπτογράφησης ZIP. Αυτή η μέθοδος είναι γρήγορη, λειτουργεί σε κάθε πλατφόρμα και είναι ο πιο απλός τρόπος για **αποθήκευση zip με κωδικό** όταν δεν χρειάζεστε την ισχυρότερη κρυπτογράφηση AES.

## Πώς να συμπιέσετε αρχεία με κωδικό
Εάν χρειάζεστε προστασία πολλαπλών αρχείων, απλώς επαναλάβετε την κλήση `CreateEntry` για κάθε αρχείο μέσα στο ίδιο μπλοκ `using`. Κάθε καταχώρηση θα κληρονομήσει τον ίδιο κωδικό, επιτρέποντάς σας να **συμπιέσετε αρχεία με κωδικό** σε ένα μόνο αρχείο.

## Πώς να κρυπτογραφήσετε zip αρχεία (παραδοσιακό vs. AES)
Η παραδοσιακή κρυπτογράφηση υποστηρίζεται ευρέως, αλλά για εξαιρετικά ευαίσθητα δεδομένα εξετάστε την επιλογή AES‑256 που προσφέρει το Aspose.Zip. Το πρότυπο κώδικα είναι το ίδιο· χρειάζεται μόνο να αντικαταστήσετε το `TraditionalEncryptionSettings` με το `AesEncryptionSettings`.

**Q: Μπορώ να κρυπτογραφήσω ένα ZIP αρχείο που υπάρχει ήδη;**
