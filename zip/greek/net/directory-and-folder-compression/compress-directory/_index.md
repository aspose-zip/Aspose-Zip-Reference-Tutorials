# Πώς να Συμπιέσετε Φάκελο Χρησιμοποιώντας το Aspose.Zip για .NET

Σε αυτό το tutorial θα ανακαλύψετε **πώς να συμπιέσετε φάκελο** γρήγορα και αξιόπιστα με το Aspose.Zip για .NET. Είτε δημιουργείτε μια επιτραπέζια εφαρμογή, μια υπηρεσία cloud‑based, είτε ένα αυτοματοποιημένο script backup, η συμπίεση ενός φακέλου σε αρχείο ZIP μπορεί να μειώσει δραστικά τις απαιτήσεις αποθήκευσης και να επιταχύνει τις μεταφορές δικτύου. Θα περάσουμε βήμα‑βήμα από κάθε στάδιο, θα εξηγήσουμε γιατί κάθε γραμμή κώδικα είναι σημαντική και θα επισημάνουμε κοινά λάθη ώστε να εφαρμόσετε την τεχνική με αυτοπεποίθηση.

## Γρήγορες Απαντήσεις
- **Τι κάνει το Aspose.Zip;** Παρέχει ένα απλό .NET API για δημιουργία και εξαγωγή αρχείων ZIP χωρίς εξωτερικές εξαρτήσεις.  
- **Πόσο χρόνο παίρνει η υλοποίηση;** Συνήθως λιγότερο από 10 λεπτά για μια βασική συμπίεση φακέλου.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, και .NET 5/6+.  
- **Χρειάζεται άδεια για παραγωγή;** Ναι, απαιτείται εμπορική άδεια για χρήση σε παραγωγικό περιβάλλον.  
- **Μπορώ να συμπιέσω πολλούς φακέλους ταυτόχρονα;** Απόλυτα—χρησιμοποιήστε τη μέθοδο `CreateEntries` σε οποιαδήποτε συλλογή `DirectoryInfo` για **συμπίεση πολλαπλών φακέλων** σε μία εκτέλεση.

## Τι σημαίνει “πώς να συμπιέσετε φάκελο”?

Η συμπίεση ενός φακέλου σημαίνει ότι παίρνετε κάθε αρχείο και υπο‑φάκελο μέσα σε έναν συγκεκριμένο κατάλογο και τα συσσωρεύετε σε ένα ενιαίο αρχείο ZIP. Αυτό μειώνει το συνολικό μέγεθος, διατηρεί την αρχική ιεραρχία και καθιστά ευκολότερη τη μεταφορά ή την αποθήκευση των δεδομένων.

## Γιατί να χρησιμοποιήσετε το Aspose.Zip για αυτήν την εργασία;

- **Ταχύτητα & Αποδοτικότητα:** Βελτιστοποιημένοι αλγόριθμοι χειρίζονται μεγάλους φακέλους γρήγορα.  
- **Καθαρά .NET:** Δεν απαιτούνται εγγενή binaries ή εργαλεία τρίτων.  
- **Πλούσιο Σύνολο Χαρακτηριστικών:** Υποστηρίζει προστασία με κωδικό (`add password zip`), streaming, και ορισμό προσαρμοσμένου επιπέδου συμπίεσης (`set compression level`).  
- **Συνεπές API:** Λειτουργεί με τον ίδιο τρόπο σε .NET Framework, .NET Core και .NET 5/6, καθιστώντας το ιδανικό για σενάρια **create zip archive .net**.

## Πώς να Συμπιέσετε Φάκελο με το Aspose.Zip

Παρακάτω υπάρχει ένα απλό παράδειγμα που δείχνει **πώς να συμπιέσετε φάκελο**. Το ίδιο μοτίβο μπορεί να επεκταθεί για **συμπίεση πολλαπλών αρχείων .net** ή για δημιουργία προσαρμοσμένων δομών αρχείων.

### Βήμα 1: Αρχικοποίηση του Καταλόγου Εγγράφων

Ορίστε τη μεταβλητή `dataDir` στη διαδρομή του καταλόγου που θέλετε να συμπιέσετε.

```csharp
string dataDir = "Your Document Directory";
```

### Βήμα 2: Δημιουργία Αρχείων Εξόδου ZIP

Ανοίξτε δύο αντικείμενα `FileStream` για τα αρχεία εξόδου ZIP. Αυτό δείχνει πώς μπορείτε να δημιουργήσετε περισσότερα από ένα αρχεία από την ίδια πηγή—χρήσιμο για εκδόσεις backup.

```csharp
using (FileStream zipFile = File.Open(dataDir + "CompressDirectory_out.zip", FileMode.Create))
{
    using (FileStream zipFile2 = File.Open(dataDir + "CompressDirectory2_out.zip", FileMode.Create))
    {
```

### Βήμα 3: Συμπίεση του Καταλόγου

Χρησιμοποιήστε την κλάση `Archive` για να προσθέσετε κάθε καταχώρηση από τον στόχο φάκελο. Το παράδειγμα χρησιμοποιεί έναν δείγμα φάκελο με όνομα **CanterburyCorpus**, αλλά μπορείτε να τον αντικαταστήσετε με οποιονδήποτε κατάλογο.

```csharp
        using (Archive archive = new Archive())
        {
            DirectoryInfo corpus = new DirectoryInfo(dataDir + "CanterburyCorpus");
            archive.CreateEntries(corpus);
            archive.Save(zipFile);
            archive.Save(zipFile2);
        }
    }
}
```

> **Συμβουλή:** Αν χρειάζεστε **create zip archive .net** με συγκεκριμένο επίπεδο συμπίεσης, ορίστε `archive.CompressionLevel` πριν καλέσετε το `Save`.
