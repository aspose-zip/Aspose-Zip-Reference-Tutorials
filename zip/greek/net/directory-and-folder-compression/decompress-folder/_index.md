# Πώς να Συμπιέσετε Φάκελο – Συμπίεση Καταλόγου με Aspose.Zip για .NET

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “compress directory to zip”;** Σημαίνει τη μετατροπή των περιεχομένων ενός φακέλου σε ένα ενιαίο .zip αρχείο.  
- **Πώς να εξάγω zip σε κατάλογο;** Χρησιμοποιήστε τη μέθοδο `Archive.ExtractToDirectory` όπως φαίνεται στον οδηγό.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** Όλες οι σύγχρονες εκδόσεις .NET Framework, .NET Core και .NET 5/6+.  
- **Απαιτείται άδεια για παραγωγή;** Ναι, απαιτείται εμπορική άδεια Aspose.Zip για μη‑δοκιμαστική χρήση.  
- **Μπορώ να το αυτοματοποιήσω σε CI/CD pipelines;** Απόλυτα—απλώς προσθέστε τον ίδιο κώδικα στα scripts κατασκευής.

## Τι είναι το “compress directory to zip”;
**Compress directory to zip** σημαίνει απλώς τη λήψη κάθε αρχείου και υπο‑φακέλου μέσα σε έναν κατάλογο και τη συσκευασία τους σε ένα ενιαίο συμπιεσμένο αρχείο. Αυτό μειώνει το μέγεθος αποθήκευσης, επιταχύνει τις μεταφορές και δημιουργεί ένα φορητό πακέτο για ανάπτυξη.

## Γιατί να χρησιμοποιήσετε Aspose.Zip για .NET;
Το Aspose.Zip παρέχει ένα **pure‑managed** API που δεν απαιτεί εγγενή DLLs, υποστηρίζει τεράστια αρχεία και διαχειρίζεται ειδικές περιπτώσεις όπως **προστασία κωδικού πρόσβασης σε zip αρχείο** και ονόματα αρχείων Unicode αυτόματα. Επίσης είναι βελτιστοποιημένο για απόδοση, καθιστώντας το ιδανικό όταν χρειάζεται να συμπιέσετε φάκελο προγραμματιστικά σε σενάρια υψηλής απόδοσης.

## Προαπαιτούμενα
- **Βιβλιοθήκη Aspose.Zip για .NET** εγκατεστημένη (κατεβάστε την [εδώ](https://releases.aspose.com/zip/net/)).  
- Ένας φάκελος στον δίσκο που θέλετε να αρχειοθετήσετε – ορίστε τη διαδρομή του στη μεταβλητή `dataDir`.  
- Περιβάλλον ανάπτυξης .NET (Visual Studio, VS Code ή οποιοδήποτε IDE προτιμάτε).

## Εισαγωγή Namespaces
Πρώτα, φέρετε τα απαιτούμενα namespaces στο πεδίο ορατότητας:

```csharp
using Aspose.Zip;
using System.IO;
```

## compress directory to zip – Οδηγός βήμα‑βήμα

### Βήμα 1: Συμπίεση φακέλου προγραμματιστικά
Θα δημιουργήσουμε ένα αρχείο zip από τον κατάλογο που σκοπεύετε να αποσυμπιέσετε αργότερα. Η βοηθητική μέθοδος `CompressDirectory.Run()` κάνει τη βαριά δουλειά.

```csharp
string dataDir = "Your Document Directory";
CompressDirectory.Run();
```

> **Συμβουλή:** Το παράδειγμα `CompressDirectory` συσκευάζει κάθε αρχείο στο `dataDir` στο `CompressDirectory_out.zip`. Μπορείτε να μετονομάσετε το αρχείο εξόδου ώστε να ταιριάζει με τις προτιμήσεις ονομασίας σας.

### Βήμα 2: extract zip to directory – Πώς να αποσυμπιέσετε φάκελο σε .NET

#### Βήμα 2.1: Άνοιγμα του αρχείου Zip
Ανοίξτε το δημιουργημένο αρχείο με ένα `FileStream`. Αυτό προετοιμάζει το αρχείο για ανάγνωση.

```csharp
using (FileStream zipFile = File.Open(dataDir + "CompressDirectory_out.zip", FileMode.Open))
{
```

#### Βήμα 2.2: Δημιουργία αντικειμένου Archive
Δημιουργήστε το αντικείμενο `Archive`, το οποίο αντιπροσωπεύει το κοντέινερ zip.

```csharp
    using (var archive = new Archive(zipFile))
    {
```

#### Βήμα 2.3: extract zip archive .net
Τέλος, εξάγετε τα περιεχόμενα σε έναν νέο φάκελο. Αυτό είναι το βήμα **extract zip to directory**.

```csharp
        archive.ExtractToDirectory(dataDir + "DecompressFolder_out");
    }
}
```

## Συνηθισμένα προβλήματα & λύσεις
| Σύμπτωμα | Πιθανή Αιτία | Διόρθωση |
|---------|--------------|----------|
| `UnauthorizedAccessException` κατά την εξαγωγή | Ο φάκελος προορισμού είναι μόνο για ανάγνωση ή σε χρήση | Βεβαιωθείτε ότι η διαδρομή προορισμού είναι εγγράψιμη και δεν είναι κλειδωμένη |
| Κενός φάκελος εξόδου μετά την εξαγωγή | Λάθος διαδρομή zip πηγής | Ελέγξτε ξανά ότι το `dataDir + "CompressDirectory_out.zip"` δείχνει στο σωστό αρχείο |
| Μεγάλα αρχεία προκαλούν OutOfMemoryException | Χρήση προεπιλεγμένου μεγέθους buffer σε πολύ μεγάλα αρχεία | Χρησιμοποιήστε `ArchiveOptions` για να αυξήσετε το μέγεθος buffer ή να μεταφέρετε τα αρχεία σε τμήματα |
