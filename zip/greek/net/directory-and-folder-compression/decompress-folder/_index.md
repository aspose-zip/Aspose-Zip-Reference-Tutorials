# Πώς να Συμπιέσετε Φάκελο – Συμπίεση Καταλόγου με Aspose.Zip για .NET

Αν ψάχνετε για μια σαφή λύση **how to zip folder** σε εφαρμογή .NET, βρίσκεστε στο σωστό σημείο. Σε αυτό το tutorial θα περάσουμε από όλη τη διαδικασία—πρώτα θα **compress directory to zip**, μετά θα σας δείξουμε τα ακριβή βήματα για **extract zip to directory** (δηλαδή how to unzip folder). Στο τέλος θα έχετε ένα επαναχρησιμοποιήσιμο, προγραμματιστικό μοτίβο για λειτουργίες zip folder που λειτουργεί σε .NET Framework, .NET Core και .NET 5/6+.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “compress directory to zip”;** Σημαίνει τη μετατροπή των περιεχομένων ενός φακέλου σε ένα ενιαίο αρχείο .zip.  
- **Πώς να εξάγω zip σε κατάλογο;** Χρησιμοποιήστε τη μέθοδο `Archive.ExtractToDirectory` όπως φαίνεται στον οδηγό.  
- **Ποιες εκδόσεις του .NET υποστηρίζονται;** Όλες οι σύγχρονες εκδόσεις του .NET Framework, .NET Core και .NET 5/6+.  
- **Απαιτείται άδεια για παραγωγή;** Ναι, απαιτείται εμπορική άδεια Aspose.Zip για χρήση εκτός δοκιμής.  
- **Μπορώ να το αυτοματοποιήσω σε CI/CD pipelines;** Απόλυτα—απλώς προσθέστε τον ίδιο κώδικα στα scripts κατασκευής.

## Τι είναι το “how to zip folder”;
**How to zip folder** σημαίνει απλώς η λήψη κάθε αρχείου και υποφακέλου μέσα σε έναν κατάλογο και η συσκευασία τους σε ένα ενιαίο συμπιεσμένο αρχείο. Αυτό μειώνει το μέγεθος αποθήκευσης, επιταχύνει τις μεταφορές και δημιουργεί ένα φορητό πακέτο για ανάπτυξη.

## Γιατί να χρησιμοποιήσετε το Aspose.Zip για .NET;
Το Aspose.Zip παρέχει ένα **pure‑managed** API που δεν απαιτεί εγγενή DLLs, υποστηρίζει τεράστια αρχεία και διαχειρίζεται ειδικές περιπτώσεις όπως **zip archive password protection** και ονόματα αρχείων Unicode αυτόματα. Είναι επίσης βελτιστοποιημένο για απόδοση, καθιστώντας το ιδανικό όταν χρειάζεται να zip folder προγραμματιστικά σε σενάρια υψηλής ροής.

## Προαπαιτούμενα
- Βιβλιοθήκη **Aspose.Zip for .NET** εγκατεστημένη (κατεβάστε την [εδώ](https://releases.aspose.com/zip/net/)).  
- Ένας φάκελος στο δίσκο που θέλετε να αρχειοθετήσετε – ορίστε τη διαδρομή του στη μεταβλητή `dataDir`.  
- Περιβάλλον ανάπτυξης .NET (Visual Studio, VS Code ή οποιοδήποτε IDE προτιμάτε).

## Εισαγωγή Namespaces
Πρώτα, φέρτε τα απαιτούμενα namespaces στο πεδίο ορατότητας:

```csharp
using Aspose.Zip;
using System.IO;
```

## Οδηγός Βήμα‑βήμα

### Βήμα 1: Συμπίεση Καταλόγου σε Zip (zip folder προγραμματιστικά)
Θα δημιουργήσουμε ένα αρχείο zip από τον κατάλογο που σκοπεύετε να αποσυμπιέσετε αργότερα. Η βοηθητική μέθοδος `CompressDirectory.Run()` κάνει το σκληρό έργο.

```csharp
string dataDir = "Your Document Directory";
CompressDirectory.Run();
```

> **Pro tip:** Το δείγμα `CompressDirectory` πακετάρει κάθε αρχείο στο `dataDir` στο `CompressDirectory_out.zip`. Μπορείτε ελεύθερα να μετονομάσετε το αρχείο εξόδου ώστε να ταιριάζει με τις ονομαστικές σας προτιμήσεις.

### Βήμα 2: Αποσυμπίεση Φακέλου – Πώς να Αποσυμπιέσετε Φάκελο σε .NET

#### Βήμα 2.1: Άνοιγμα του Αρχείου Zip
Ανοίξτε το παραγόμενο αρχείο με ένα `FileStream`. Αυτό προετοιμάζει το αρχείο για ανάγνωση.

```csharp
using (FileStream zipFile = File.Open(dataDir + "CompressDirectory_out.zip", FileMode.Open))
{
```

#### Βήμα 2.2: Δημιουργία Αντικειμένου Archive
Δημιουργήστε το αντικείμενο `Archive`, το οποίο αντιπροσωπεύει το κοντέινερ zip.

```csharp
    using (var archive = new Archive(zipFile))
    {
```

#### Βήμα 2.3: Εξαγωγή σε Κατάλογο
Τέλος, εξάγετε τα περιεχόμενα σε έναν νέο φάκελο. Αυτό είναι το βήμα **extract zip to directory**.

```csharp
        archive.ExtractToDirectory(dataDir + "DecompressFolder_out");
    }
}
```

- **Συνέπεια:** Η χρήση της ίδιας βιβλιοθήκης για συμπίεση και εξαγωγή εγγυάται συμβατές μορφές αρχείων.  
- **Απόδοση:** Το Aspose.Zip ρέει δεδομένα αποδοτικά, έτσι ακόμη και αρχεία πολλαπλών gigabyte διαχειρίζονται με χαμηλή χρήση μνήμης.  
- **Ασφάλεια:** Η ενσωματωμένη υποστήριξη κωδικού πρόσβασης σημαίνει ότι μπορείτε να ασφαλίσετε το zip αρχείο χωρίς επιπλέον κώδικα.

## Συνηθισμένες Περιπτώσεις Χρήσης
- **Αυτοματοποιημένα αντίγραφα ασφαλείας** – zip έναν φάκελο καταγραφών κάθε νύχτα και αποθηκεύστε το σε cloud storage.  
- **Πακέτα ανάπτυξης** – ομαδοποιήστε στατικά web assets πριν τη δημοσίευση σε διακομιστή.  
- **Ανταλλαγή δεδομένων** – στείλτε μια συλλογή αρχείων μεταξύ υπηρεσιών ως ένα ενιαίο αρχείο.

## Συνηθισμένα Προβλήματα & Λύσεις
| Σύμπτωμα | Πιθανή Αιτία | Διόρθωση |
|---------|--------------|----------|
| `UnauthorizedAccessException` κατά την εξαγωγή | Ο φάκελος προορισμού είναι μόνο για ανάγνωση ή σε χρήση | Βεβαιωθείτε ότι η διαδρομή προορισμού είναι εγγράψιμη και δεν είναι κλειδωμένη |
| Κενός φάκελος εξόδου μετά την εξαγωγή | Λάθος διαδρομή zip αρχείου προέλευσης | Ελέγξτε ξανά ότι το `dataDir + "CompressDirectory_out.zip"` δείχνει στο σωστό αρχείο |
| Μεγάλα αρχεία προκαλούν OutOfMemoryException | Χρήση προεπιλεγμένου μεγέθους buffer σε πολύ μεγάλα αρχεία | Χρησιμοποιήστε `ArchiveOptions` για αύξηση του buffer ή ροή αρχείων σε τμήματα |
