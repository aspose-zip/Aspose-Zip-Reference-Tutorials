## c# create 7z archive – Step‑by‑Step Guide

### Step 1: Set the Resource Directory Path

Πριν δημιουργήσετε SevenZip entries, ορίστε τη διαδρομή προς τον φάκελο πόρων σας. Αντικαταστήστε το `"Your Document Directory"` στη μεταβλητή `dataDir` με την πραγματική διαδρομή.

```csharp
string dataDir = "Your Document Directory";
```

> **Συμβουλή:** Η χρήση απόλυτης διαδρομής εξαλείφει τη σύγχυση όταν η εφαρμογή εκτελείται από διαφορετικό τρέχον φάκελο.

### Step 2: Create SevenZip Entries (compress directory to 7z)

Τώρα, ας εμβαθύνουμε στον πυρήνα της διαδικασίας – δημιουργία SevenZip entries. Αυτό το βήμα **συμπιέζει το φάκελο σε μορφή 7z**.

```csharp
//ExStart: CreateSevenZipEntries
using (SevenZipArchive archive = new SevenZipArchive())
{
    archive.CreateEntries(dataDir);
    archive.Save("SevenZip.7z");
}
//ExEnd: CreateSevenZipEntries
```

Ο παραπάνω κώδικας αρχικοποιεί ένα `SevenZipArchive`, προσθέτει κάθε αρχείο από τον καθορισμένο φάκελο και γράφει το συμπιεσμένο αρχείο στο **SevenZip.7z**.

### Step 3: Display Success Message

Για να βεβαιωθείτε ότι όλα πήγαν ομαλά, εμφανίστε ένα μήνυμα επιτυχίας:

```csharp
Console.WriteLine("Successfully Created a Seven Zip File");
```

Τώρα έχετε ένα έτοιμο για κοινή χρήση **αρχείο 7z** που μπορεί να μεταφερθεί, αποθηκευτεί ή να υποβληθεί σε περαιτέρω επεξεργασία.
