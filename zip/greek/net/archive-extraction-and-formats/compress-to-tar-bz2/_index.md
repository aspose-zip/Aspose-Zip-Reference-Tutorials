## Τι είναι η “προσθήκη αρχείων σε tar”;
Η προσθήκη αρχείων σε ένα **tar** (Tape Archive) δημιουργεί ένα ενιαίο ασυμπίεστο κοντέινερ που διατηρεί τη δομή των καταλόγων και τα μεταδεδομένα των αρχείων. Όταν στη συνέχεια εφαρμόζετε συμπίεση Bzip2, το αποτέλεσμα είναι ένα αρχείο **tar.bz2** (TarBz2) — ιδανικό για αποδοτική αποθήκευση και μεταφορά.

## Γιατί να συμπιέσετε αρχεία σε TarBz2 με το Aspose.Zip;
- **Ταχύτητα & Απλότητα** – Κλήσεις API μιας γραμμής διαχειρίζονται τόσο τη δημιουργία tar όσο τη συμπίεση Bzip2.
- **Διαπλατφόρμα** – Λειτουργεί σε Windows, Linux και macOS .NET runtime.
- **Λεπτομερής έλεγχος** – Επιλέξτε ποια αρχεία να συμπεριλάβετε, ορίστε προσαρμοσμένα ονόματα καταχωρήσεων και ρέξτε απευθείας στο δίσκο.

## Βήμα 1: Ορισμός του Document Directory

Ορίστε τη διαδρομή που δείχνει στον φάκελο που περιέχει τα αρχεία που θέλετε να αρχειοθετήσετε.

```csharp
string dataDir = "Your Document Directory";
```

> Αντικαταστήστε το `"Your Document Directory"` με την απόλυτη ή σχετική διαδρομή προς το φάκελο προέλευσης.

## Βήμα 2: Προσθήκη αρχείων σε tar και δημιουργία αρχείου TarBz2

Ο πυρήνας της διαδικασίας είναι η δημιουργία ενός `TarArchive`, η προσθήκη καταχωρήσεων και στη συνέχεια η περιτύλιξη του με ένα `Bzip2Archive`. Ο παρακάτω κώδικας δείχνει **πώς να δημιουργήσετε tarbz2** με καθαρό, disposable‑pattern στυλ.

```csharp
//ExStart: CompressFile
using (Bzip2Archive bz2 = new Bzip2Archive())
{
    using (TarArchive archive = new TarArchive())
    {
        archive.CreateEntry("alice29.txt", dataDir + "alice29.txt");
        archive.CreateEntry("lcet10.txt", dataDir + "lcet10.txt");

        bz2.SetSource(archive);
        bz2.Save(dataDir + "archive.tar.bz2");
    }
}
```

- `CreateEntry` προσθέτει κάθε αρχείο στο **tar** κοντέινερ.  
- `bz2.SetSource(archive)` λέει στο Bzip2 archive να συμπιέσει ολόκληρο το ρεύμα tar.  
- `bz2.Save(...)` γράφει το τελικό αρχείο **tar.bz2** στο δίσκο.

**Συμβουλή:** Για **συμπίεση αρχείων σε tarbz2** μαζικά, απλώς επαναλάβετε το `archive.CreateEntry` για κάθε αρχείο που χρειάζεστε.

## Συνηθισμένα Προβλήματα & Λύσεις

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| **File not found** error | Λάθος διαδρομή `dataDir` ή λείπει η επέκταση αρχείου | Επαληθεύστε τη πλήρη διαδρομή και βεβαιωθείτε ότι το αρχείο υπάρχει. |
| **Empty archive** | Δεν έχουν προστεθεί καταχωρήσεις πριν το `bz2.Save` | Προσθέστε τουλάχιστον μία κλήση `CreateEntry`. |
| **Permission denied** | Η εφαρμογή δεν έχει δικαίωμα εγγραφής στο φάκελο εξόδου | Εκτελέστε την εφαρμογή με τα κατάλληλα δικαιώματα ή επιλέξτε φάκελο με δυνατότητα εγγραφής. |
