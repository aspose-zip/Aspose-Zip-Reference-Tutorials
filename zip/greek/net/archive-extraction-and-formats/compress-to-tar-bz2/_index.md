## Πώς να συμπιέσετε tar

Η προσθήκη αρχείων σε ένα **tar** (Tape Archive) δημιουργεί ένα ενιαίο ασυμπίεστο κοντέινερ που διατηρεί τη δομή των φακέλων και τα μεταδεδομένα των αρχείων. Όταν στη συνέχεια εφαρμόζετε συμπίεση Bzip2, το αποτέλεσμα είναι ένα **tar.bz2** (TarBz2) αρχείο—ιδανικό για αποδοτική αποθήκευση και μεταφορά. Το Aspose.Zip κάνει αυτή τη διαδικασία μια‑γραμμή, διαχειριζόμενο τόσο τη δημιουργία tar όσο και τη συμπίεση Bzip2 στο παρασκήνιο.

- **Αξιόπιστη υποστήριξη .NET** – Πλήρως συμβατό με σενάρια **tar archive .net**, από κονσόλες μέχρι web services.

## Βήμα 1: Ορίστε το Φάκελο Εγγράφων

Καθορίστε τη διαδρομή που δείχνει στον φάκελο που περιέχει τα αρχεία που θέλετε να αρχειοθετήσετε.

```csharp
string dataDir = "Your Document Directory";
```

> Αντικαταστήστε το `"Your Document Directory"` με την απόλυτη ή σχετική διαδρομή προς το φάκελο προέλευσης.

## Βήμα 2: Προσθέστε αρχεία στο tar και δημιουργήστε ένα TarBz2 αρχείο

Ο πυρήνας της διαδικασίας είναι η δημιουργία ενός `TarArchive`, η προσθήκη entries, και στη συνέχεια η περιτύλιξή του με ένα `Bzip2Archive`. Ο κώδικας παρακάτω δείχνει **πώς να δημιουργήσετε tarbz2** με καθαρό, disposable‑pattern στυλ.

```csharp
//ExStart: CompressFile
using (Bzip2Archive bz2 = new Bzip2Archive())
{
    using (TarArchive archive = new TarArchive())
    {
        archive.CreateEntry("alice29.txt", dataDir + "alice29.txt");
        archive.CreateEntry("lcet10.txt", dataDir + "lcet10.txt");

        bz2.SetSource(archive);
        bz2.Save(dataDir + "archive.tar.bz2");
    }
}
```

- `CreateEntry` προσθέτει κάθε αρχείο στο **tar** κοντέινερ.  
- `bz2.SetSource(archive)` λέει στο Bzip2 archive να συμπιέσει ολόκληρο το ρεύμα tar.  
- `bz2.Save(...)` γράφει το τελικό **tar.bz2** αρχείο στο δίσκο.

**Συμβουλή:** Για **συμπίεση αρχείων σε tarbz2** μαζικά, απλώς επαναλάβετε `archive.CreateEntry` για κάθε αρχείο που χρειάζεστε.

## Συνηθισμένα Προβλήματα & Λύσεις

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| **File not found** error | Λάθος διαδρομή `dataDir` ή λείπει η επέκταση αρχείου | Επαληθεύστε τη πλήρη διαδρομή και βεβαιωθείτε ότι το αρχείο υπάρχει. |
| **Empty archive** | Δεν έχουν προστεθεί entries πριν το `bz2.Save` | Προσθέστε τουλάχιστον μία κλήση `CreateEntry`. |
| **Permission denied** | Η εφαρμογή δεν έχει δικαίωμα εγγραφής στον φάκελο εξόδου | Εκτελέστε την εφαρμογή με τα κατάλληλα δικαιώματα ή επιλέξτε φάκελο με δυνατότητα εγγραφής. |
