# Πώς να εξάγετε αρχείο Zip με κωδικό πρόσβασης χρησιμοποιώντας το Aspose.Zip για .NET

Σε σύγχρονες εφαρμογές .NET, η προστασία ευαίσθητων δεδομένων μέσα σε αρχεία ZIP είναι συχνή απαίτηση. Αυτό το tutorial δείχνει **πώς να εξάγετε zip με κωδικό πρόσβασης** όταν κάθε καταχώρηση χρησιμοποιεί διαφορετικό κωδικό, παρέχοντάς σας λεπτομερή έλεγχο ασφαλείας ενώ διατηρεί τη διαδικασία εξαγωγής απλή. Ακολουθώντας αυτό το παράδειγμα Aspose.Zip θα δείτε ακριβώς πώς να πραγματοποιήσετε εξαγωγή zip με κωδικό πρόσβασης για μεμονωμένες καταχωρήσεις.

## Γρήγορες Απαντήσεις
- **Ποια βιβλιοθήκη πρέπει να χρησιμοποιήσω;** Aspose.Zip για .NET.  
- **Μπορώ να εξάγω καταχωρήσεις που έχουν διαφορετικούς κωδικούς;** Ναι—κάθε καταχώρηση μπορεί να ανοιχθεί με τον δικό της κωδικό.  
- **Χρειάζεται άδεια για παραγωγή;** Απαιτείται εμπορική άδεια· διατίθεται δωρεάν δοκιμή.  
- **Υποστηριζόμενες πλατφόρμες;** .NET Framework, .NET Core, .NET 5/6+.  
- **Τυπικός χρόνος υλοποίησης;** Περίπου 10 λεπτά για ένα βασικό σενάριο.

## Τι είναι το “πώς να εξάγετε zip”;
Η εξαγωγή ενός αρχείου ZIP σημαίνει ανάγνωση του συμπιεσμένου περιεχομένου και εγγραφή του στο σύστημα αρχείων. Όταν το αρχείο είναι προστατευμένο με κωδικό, πρέπει επίσης να παρέχετε τον σωστό κωδικό για κάθε καταχώρηση πριν τα δεδομένα αποσυμπιεστούν.

## Γιατί να χρησιμοποιήσετε το Aspose.Zip για εξαγωγή με κωδικό πρόσβασης;
- **Λεπτομερής ασφάλεια:** Κάθε αρχείο μπορεί να έχει τον δικό του κωδικό, μειώνοντας τον κίνδυνο εάν ένας κωδικός παραβιαστεί.  
- **Ευελιξία:** Μπορείτε να αποφασίσετε προγραμματιστικά ποιος κωδικός θα εφαρμοστεί βάσει επιχειρηματικής λογικής (π.χ., ρόλοι χρηστών).  
- **Απόδοση:** Το Aspose.Zip επεξεργάζεται τις καταχωρήσεις στη μνήμη, αποφεύγοντας την ανάγκη αποσυμπίεσης ολόκληρου του αρχείου πρώτα.  
- **Υποστήριξη πολλαπλών πλατφορμών:** Λειτουργεί σε Windows, Linux και macOS με .NET 5/6+.

## Προαπαιτούμενα

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε:

- **Aspose.Zip για .NET** εγκατεστημένο στο έργο σας. Μπορείτε να βρείτε την επίσημη τεκμηρίωση [εδώ](https://reference.aspose.com/zip/net/).  
- Περιβάλλον ανάπτυξης .NET (Visual Studio, Rider ή VS Code) που στοχεύει .NET 5 ή νεότερο.  
- Ένα αρχείο ZIP που περιέχει καταχωρήσεις κρυπτογραφημένες με **διαφορετικούς κωδικούς** (το δείγμα που χρησιμοποιείται εδώ είναι `different_password.zip`).

## Εισαγωγή Namespaces

Πρώτα, εισάγετε τα namespaces που απαιτούνται για εργασία με αρχεία:

```csharp
using Aspose.Zip;
using System.IO;
```

Αυτές οι δύο δηλώσεις `using` σας δίνουν πρόσβαση στην κλάση `Archive` και στα τυπικά εργαλεία I/O.

## Βήμα 1: Ορισμός του Καταλόγου Εργασίας

Ορίστε το φάκελο όπου βρίσκεται το αρχείο ZIP και όπου θα γραφτούν τα εξαγόμενα αρχεία:

```csharp
string dataDir = "Your Document Directory";
```

> **Συμβουλή:** Χρησιμοποιήστε το `Path.Combine` για δημιουργία διαδρομών ανεξαρτήτως πλατφόρμας εάν χρειάζεται να υποστηρίξετε Linux/macOS.

## Πώς να εξάγετε Zip με κωδικό πρόσβασης χρησιμοποιώντας το Aspose.Zip

Παρακάτω περπατάμε βήμα-βήμα την ακριβή διαδικασία ανοίγματος του αρχείου και εξαγωγής κάθε καταχώρησης με τον δικό της κωδικό. Αυτή η ενότητα δείχνει **εξαγωγή zip με κωδικό πρόσβασης** για κάθε καταχώρηση, που αποτελεί τον πυρήνα της διαδικασίας “πώς να εξάγετε zip”.

### Βήμα 2: Εξαγωγή καταχωρήσεων αρχείου με διαφορετικούς κωδικούς

#### Βήμα 2.1: Άνοιγμα του αρχείου Zip

```csharp
using (FileStream zipFile = File.Open(dataDir + "\\different_password.zip", FileMode.Open))
{
    using (Archive archive = new Archive(zipFile))
    {
        // Continue to the next steps...
    }
}
```

Το αντικείμενο `Archive` αντιπροσωπεύει το κοντέινερ ZIP. Η διατήρηση του `FileStream` και του `Archive` μέσα σε μπλοκ `using` εξασφαλίζει ότι όλοι οι πόροι απελευθερώνονται άμεσα.

#### Βήμα 2.2: Εξαγωγή της πρώτης καταχώρησης (Κωδικός = “first_pass”)

```csharp
archive.Entries[0].Extract(dataDir + "alice29_extracted_pass_out.txt", "first_pass");
```

Εδώ **εξάγουμε πολλαπλές καταχωρήσεις zip** προσπερνώντας τη συλλογή `Entries`. Η πρώτη καταχώρηση αποκρυπτογραφείται με τον κωδικό `"first_pass"`.

#### Βήμα 2.3: Εξαγωγή της δεύτερης καταχώρησης (Κωδικός = “second_pass”)

```csharp
archive.Entries[1].Extract(dataDir + "asyoulik_extracted_pass_out.txt", "second_pass");
```

Η δεύτερη καταχώρηση χρησιμοποιεί διαφορετικό κωδικό, επιδεικνύοντας τη διαχείριση **κωδικού εξαγωγής zip entry** για κάθε μεμονωμένο αρχείο.

#### Βήμα 2.4: (Προαιρετικό) Επανάληψη σε όλες τις καταχωρήσεις

Εάν χρειάζεται να **εξάγετε πολλαπλές καταχωρήσεις zip** χωρίς σκληρή κωδικοποίηση δεικτών, μπορείτε να επαναλάβετε το `archive.Entries` και να παρέχετε τον κατάλληλο κωδικό για κάθε καταχώρηση βάσει της δικής σας λογικής αναζήτησης. Αυτό το μοτίβο κλιμακώνεται καλά όταν εργάζεστε με μεγάλα αρχεία.

| Πρόβλημα | Αιτία | Διόρθωση |
