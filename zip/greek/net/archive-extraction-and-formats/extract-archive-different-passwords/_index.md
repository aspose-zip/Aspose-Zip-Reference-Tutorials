# Πώς να εξάγετε αρχείο Zip με κωδικό πρόσβασης χρησιμοποιώντας το Aspose.Zip για .NET

## Γρήγορες απαντήσεις
- **Ποια βιβλιοθήκη πρέπει να χρησιμοποιήσω;** εξάγω καταχωρήσεις που έχουν διαφορετικούς κωδικούς πρόσβασης;**ικό της κωδικό.  
- **Χρειάζομαι άδεια για παραγωγή;** Απαιτείται εμπορική άδεια· διατίθεται δωρεάν δοκιμή.  
- **Υποστηριζόμενες πλατφόρμες;** .NET Framework, .NET Core, .NET 5/6+.  
- **Τυπικός χρόνος υλοποίησης;** Περίπου 10 λεπτά για ένα βασικό σενάριο.  

## Προαπαιτούμενα

Πριν ξεκινήσουμε, ββαιωθείτε ότι έχετε:

- **Aspose.Zip for .NET** εγκατεστημένο στο έργο σας. Μπορείτε να βρείτε την επίσημη τεκμηρίωση [εδώ](https://reference.aspose.com/zip/net/).
- Ένα περιβάλλον ανάπτυξης .NET (Visual Studio, Rider ή VS Code) που στοχεύει στο .NET 5 ή νεότερο.
- Ένα αρχείο ZIP που περιέχει καταχωρήσεις κρυπτογραφημένες με **διαφορετικούς κωδικούς πρόσβασης** (το δείγμα που χρησιμοποιείται εδώ είναι `different_password.zip`).

## Εισαγωγήνόματα χώρων που απαιτούν```csharp
using Aspose.Zip;
using System.IO;
```

Αυτές οι δύο δηλώσεις `using` σας δίνουν πρόσβαση στην κλάση `Archive` και στα τυπικά εργαλεία I/O.

##Παρακάτω θα περάσουμε από τη διαδικασία, από τον εντοπισμό του αρχείου έως την εξαγωγή κάθε καταχώρησης με τον μοναδικό της κωδικό. Αυτός ο οδηγός βήμα‑βήμα εξασφαλίζει ότι δεν θα χάσετε καμία λεπτομέρ .net**.

## Βήμα 1: Ορισμός του καταλόγ φάκελο όπου βρίσκεται το αρχείο ZIP και όπου θα γραφτούν τα εξαγόμενα αρχεία:

```csharp
string dataDir = "Your Document Directory";
```

> **Συμβουλή:** Χρησιμοποιήστε `Path.Combine` για δημιουργία διαδρομών δια‑πλατφόρμας εάν χρειάζεται να υποστηρίξετε Linux/macOS.

## Βήμα 2: Εξαγωγή καταχωρήσεων αρχείου με διαφορετικούς κωδικούς πρόσβασης

### Βήμα 2.1: Άνοιγμα του αρχείου Zip

```csharp
using (FileStream zipFile = File.Open(dataDir + "\\different_password.zip", FileMode.Open))
{
    using (Archive archive = new Archive(zipFile))
    {
        // Continue to the next steps...
    }
}
```

Το αντικείμενο `Archive` αντιπροσωπεύει το δοχείο ZIP. Η διατήρηση του `FileStream` και του `Archive` μέσα σε μπλοκ `using` εξασφαλίζει ότι όλοι οι πόροι απελευθερώνονται άμεσα.

### Βήμα 2.2: Εξαγωγή της πρώτης καταχώρησης (Κωδικός = “first_pass”)

```csharp
archive.Entries[0].Extract(dataDir + "alice29_extracted_pass_out.txt", "first_pass");
```

Εδώ **εξάγουμε πολλαπλές καταχωρήσεις zip** προσπελαύνοντάς τες μέσω της συλλογής `Entries`. Η πρώτη καταχώρηση αποκρυπτογραφείται με τον κωδικό `"first_pass"`.

### Βήμα 2.3: Εξαγωγή της δεύτερης καταχώρησης (Κωδικός = “second_pass”)

```csharp
archive.Entries[1].Extract(dataDir + "asyoulik_extracted_pass_out.txt", "second_pass");
```

Η δεύτερη καταχώρηση χρησιμοποιεί διαφορετικό κωδικό, επιδεικνύοντας **εξαγωγή zip προστατευμένου με κωδικό** για κάθε μεμονωμένο αρχείο.

## Γιατί αυτή η προσέγγιση είναι σημαντική

- **Αναλυτική ασφάλεια:** Κάθε αρχείο μπορεί να έχει τον δικό του κωδικό, μειώνοντας τον κίνδυνο εάν ένας κωδικός παραβιαστεί.  
- **Ευελιξία:** Μπορείτε προγραμματιστικά να αποφασίσετε ποιον κωδικό να εφαρμόσετε βάσει επιχειρηματικής λογικής (π.χ., ρόλοι χρηστών).  
- **Απόδοση:** Το Aspose.Zip επεξεργάζεται τις καταχωρήσεις στη μνήμη, αποφεύγοντας την ανάγκη αποσυμπίεσης ολόκληρου του αρχείου πρώτα.  

## Συνηθισμένα προβλήματα & λύσεις

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| *“Invalid password” exception* | Λάθος κωδικός παρεχόμενος ή η καταχώρηση δεν είναι κρυπτογραφημένη. | Επαληθεύστε το κείμενο του κωδικού και βεβαιωθείτε ότι η καταχώρηση είναι προστατευμένη με κωδικό. |
| *File not found* | Η διαδρομή `dataDir` είναι λανθασμένη. | Χρησιμοποιήστε `Path.Combine(dataDir, "different_password.zip")` και ελέγξτε ξανά το φάκελο. |
| *Large archives cause high memory usage* | Όλες οι καταχωρήσεις φορτώνονται στη μνήμη από προεπιλογή. | Μεταδώστε κάθε καταχώρηση ξεχωριστά ή χρησιμοποιήστε `Archive.ExtractToDirectory` με κλήση κωδικού (αν υποστηρίζεται). |
