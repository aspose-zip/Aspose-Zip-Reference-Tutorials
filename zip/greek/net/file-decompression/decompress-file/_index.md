## Βήμα 1: Εξαγωγή Αρχείου Lzip C# (extract lzip archive c#)

Ο πυρήνας της λειτουργίας **c# extract from archive** είναι ένα σύντομο μπλοκ `using` που ανοίγει το αρχείο Lzip και γράφει τα αποσυμπιεσμένα δεδομένα σε νέο αρχείο.

```csharp
//ExStart: OpenLzipArchive
using (var archive = new LzipArchive(Path.Combine(dataDir, "archive.lz")))
{
    using (var extracted = File.Create(Path.Combine(dataDir, "output.txt")))
    {
        archive.Extract(extracted);
    }
}
//ExEnd: OpenLzipArchive
Console.WriteLine("Successfully Opened Lzip Archive");
```

Αυτό το απόσπασμα δείχνει το πρότυπο **extract lzip archive c#**:

1. **Create** ένα αντικείμενο `LzipArchive` που δείχνει στο αρχείο προέλευσης.  
2. **Create** το αρχείο προορισμού (`output.txt`).  
3. **Call** `Extract` για να γράψετε τα αποσυμπιεσμένα bytes.  
4. Οι δηλώσεις `using` εγγυώνται ότι όλα τα streams κλείνουν αυτόματα.
