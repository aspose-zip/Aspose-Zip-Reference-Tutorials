## Τι είναι ένα αρχείο WIM;

Ένα **WIM (Windows Imaging Format)** είναι ένα αρχείο που αποθηκεύει μία ή περισσότερες εικόνες δίσκου σε ένα ενιαίο, συμπιεσμένο αρχείο. Χρησιμοποιείται ευρέως από το Windows Setup, το DISM και λύσεις ανάπτυξης. Κάθε εικόνα μέσα σε ένα WIM μπορεί να αντιμετωπιστεί ως εικονικό σύστημα αρχείων, κάτι που καθιστά πολύ χρήσιμη την επιλεκτική εξαγωγή.

## Γιατί να χρησιμοποιήσετε το Aspose.Zip για .NET;

Το Aspose.Zip προσφέρει ένα καθαρά διαχειριζόμενο,跨平台 API που εξαλείφει την ανάγκη για εγγενείς εξαρτήσεις. Υποστηρίζει:

* Άμεση πρόσβαση σε μεμονωμένες εικόνες μέσα σε ένα αρχείο WIM.  
* Λειτουργίες βασισμένες σε ροές (stream) που διατηρούν τη χρήση μνήμης χαμηλή.  
* Απρόσκοπτη ενσωμάτωση τόσο σε έργα .NET Framework όσο και .NET Core.  

Αυτά τα χαρακτηριστικά σας επιτρέπουν να δημιουργήσετε αξιόπιστα εργαλεία εξαγωγής χωρίς να ανησυχείτε για ειδικές ιδιαιτερότητες πλατφόρμας.

## Πώς να Εξάγετε WIM σε Φάκελο

Παρακάτω θα βρείτε τα ακριβή βήματα που δείχνουν **πώς να εξάγετε αρχεία WIM** χρησιμοποιώντας το Aspose.Zip. Ακολουθήστε προσεκτικά κάθε βήμα.

### Βήμα 1: Ορίστε τον Κατάλογο Εγγράφου σας

Ορίστε τη διαδρομή του καταλόγου όπου βρίσκεται το αρχείο WIM. Αντικαταστήστε το `"Your Document Directory"` με την πραγματική διαδρομή του φακέλου σας.

```csharp
string dataDir = "Your Document Directory";
```

### Βήμα 2: Αποσυμπιέστε το Αρχείο WIM

Ανοίξτε το αρχείο WIM χρησιμοποιώντας ένα `FileStream` και στη συνέχεια εξάγετε τα περιεχόμενα της πρώτης εικόνας σε έναν προορισμό.

```csharp
using (FileStream fs = File.OpenRead(dataDir + "corpus.wim"))
{
    using (WimArchive archive = new WimArchive(fs))
    {
        archive.Images[0].ExtractToDirectory(dataDir + "DecompressWim_out");
    }
}
```

Το απόσπασμα διαβάζει το αρχείο WIM, προσπελάζει την πρώτη εικόνα (`Images[0]`) και γράφει όλα τα αρχεία στο **DecompressWim_out**. Μπορείτε να αλλάξετε το δείκτη εάν θέλετε να εξάγετε διαφορετική εικόνα όταν το αρχείο περιέχει πολλές.
