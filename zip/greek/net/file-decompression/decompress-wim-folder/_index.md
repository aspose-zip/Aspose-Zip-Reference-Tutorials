## Πώς να Εξάγετε WIM σε Φάκελο

Παρακάτω θα βρείτε τα ακριβή βήματα που δείχνουν **πώς να εξάγετε** αρχεία WIM χρησιμοποιώντας το Aspose.Zip. Ακολουθήστε προσεκτικά κάθε βήμα.

### Βήμα 1: Ορίστε τον Κατάλογο Εγγράφων σας

Ορίστε τη διαδρομή του καταλόγου όπου βρίσκεται το αρχείο Wim. Αντικαταστήστε το `"Your Document Directory"` με την πραγματική διαδρομή του καταλόγου εγγράφων σας.

```csharp
string dataDir = "Your Document Directory";
```

### Βήμα 2: Αποσυμπίεση Αρχείου Wim

Ανοίξτε το αρχείο Wim χρησιμοποιώντας ένα `FileStream` και στη συνέχεια εξάγετε τα περιεχόμενα σε έναν καθορισμένο φάκελο χρησιμοποιώντας το Aspose.Zip.

```csharp
using (FileStream fs = File.OpenRead(dataDir + "corpus.wim"))
{
    using (WimArchive archive = new WimArchive(fs))
    {
        archive.Images[0].ExtractToDirectory(dataDir + "DecompressWim_out");
    }
}
```

Αυτό το απόσπασμα κώδικα διαβάζει το αρχείο Wim, προσπελαύνει την πρώτη του εικόνα και εξάγει τα περιεχόμενα στον καθορισμένο φάκελο εξόδου.
