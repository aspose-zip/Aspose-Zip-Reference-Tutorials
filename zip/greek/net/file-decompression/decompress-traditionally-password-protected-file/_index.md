# εξαγωγή zip με κωδικό πρόσβασης χρησιμοποιώντας Aspose.Zip για .NET

## Γρήγορες Απαντήσεις
- **Ποια είναι η κύρια κλάση για τη διαχείριση αρχείων zip;** `Archive` από το namespace Aspose.Zip.  
- **Ποια μέθοδος παρέχει τον κωδικό πρόσβασης;** Περνάτε το `DecryptionPassword` μέσω του `ArchiveLoadOptions`.  
- **Μπορώ να αποσυμπιέσω zip προστατευμένο με κωδικό σε .NET Core;** Ναι, το Aspose.Zip υποστηρίζει .NET Framework, .NET Core και .NET 5/6+.

- **Πόσες γραμμές κώδικα χρειάζονται;** Λιγότερες από 20 γραμμές (χωρίς τις δηλώσεις using).

## Βήμα 2: Αποσυμπίεση Παραδοσιακά Προστατευμένου με Κωδικό Αρχείου
Τώρα ας εξάγουμε το περιεχόμενο. Ο κώδικας παρακάτω δείχνει ακριβώς πώς να **c# unzip password protected** αρχεία χρησιμοποιώντας Aspose.Zip:

```csharp
// ExStart: DecompressTraditionallyPasswordProtectedFile
using (FileStream fs = File.OpenRead(dataDir + "CompressWithTraditionalEncryption_out.zip"))
{
    using (var extracted = File.Create(dataDir + "alice_extracted_out.txt"))
    {
        using (Archive archive = new Archive(fs, new ArchiveLoadOptions() { DecryptionPassword = "p@s$" }))
        {
            using (var decompressed = archive.Entries[0].Open())
            {
                byte[] b = new byte[8192];
                int bytesRead;
                while (0 < (bytesRead = decompressed.Read(b, 0, b.Length)))
                {
                    extracted.Write(b, 0, bytesRead);
                }
            }
        }
    }
}
}
// ExEnd: DecompressTraditionallyPasswordProtectedFile
```

Σε αυτό το snippet κάνουμε:

1. Ανοίγουμε το κρυπτογραφημένο αρχείο ZIP ως ροή μόνο για ανάγνωση.  
2. Δημιουργούμε ένα νέο αρχείο (`alice_extracted_out.txt`) όπου θα γραφούν τα αποσυμπιεσμένα δεδομένα.  
3. Δημιουργούμε ένα `Archive` με `ArchiveLoadOptions`, περνώντας τον κωδικό (`"p@s$"`).  
4. Πρόσβαση στην πρώτη καταχώρηση του αρχείου (υποθέτοντας ένα μόνο αρχείο) και αντιγραφή των bytes της στο αρχείο εξόδου.

Όταν ο κώδικας ολοκληρωθεί, θα έχετε επιτυχώς **extract zip with password** και θα λάβετε το αρχικό περιεχόμενο του αρχείου.
