# Πώς να Αποσυμπιέσετε Αρχεία με το Aspose.Zip για .NET

## Εισαγωγή

Όταν χρειάζεται να **πώς να αποσυμπιέσετε αρχεία** γρήγορα και αξιόπιστα σε περιβάλλον .NET, το Aspose.Zip για .NET προσφέρει ένα καθαρό, υψηλής απόδοσης API που εξαλείφει την ταλαιπωρία της χειροκίνητης εξαγωγής. Είτε εργάζεστε με ένα μόνο αρχείο, μια δέσμη αρχείων καταγραφής ή ένα zip με προστασία κωδικού, αυτός ο οδηγός σας δείχνει ακριβώς πώς να εξάγετε τα περιεχόμενα ενός φακέλου zip και να διαχειριστείτε κρυπτογραφημένα αρχεία με λίγες μόνο γραμμές κώδικα C#.

## Γρήγορες Απαντήσεις
- **Τι κάνει το Aspose.Zip για .NET;** Παρέχει ένα απλό API για δημιουργία, ανάγνωση και εξαγωγή αρχείων ZIP, TAR, GZIP και άλλων μορφών αρχείων σε C#.
- **Μπορώ να αποσυμπιέσω πολλαπλά αρχεία ταυτόχρονα;** Ναι, η βιβλιοθήκη σας επιτρέπει να εξάγετε όλες τις καταχωρήσεις με μία κλήση ή να τις επεξεργαστείτε ξεχωριστά.
- **Υποστηρίζεται η εξαγωγή με προστασία κωδικού;** Απόλυτα – μπορείτε να δώσετε έναν κωδικό για να ξεκλειδώσετε κρυπτογραφημένα αρχεία.
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7 και μεταγενέστερες.
- **Χρειάζεται άδεια για ανάπτυξη;** Μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση· απαιτείται εμπορική άδεια για χρήση σε παραγωγή.

## Πώς να Αποσυμπιέσετε Αρχεία Χρησιμοποιώντας το Aspose.Zip για .NET
Η αποσυμπίεση αρχείων με το Aspose.Zip είναι απλή. Η βιβλιοθήκη παρέχει τη μέθοδο `Extract`, η οποία μπορεί να στοχεύει σε έναν φάκελο, ένα συγκεκριμένο αρχείο ή ένα ρεύμα μνήμης. Παρακάτω περιγράφουμε τη συνήθη ροή εργασίας:

1. **Δημιουργήστε μια παρουσία `Archive`** που δείχνει στο αρχείο zip σας.  
2. **Καλέστε το `Extract`** με τον προορισμό φακέλου.  
3. **Προαιρετικά περάστε έναν κωδικό** εάν το αρχείο είναι κρυπτογραφημένο (`extract password protected zip`).  

Αυτή η προσέγγιση λειτουργεί για οποιονδήποτε τύπο αρχείου που υποστηρίζεται από το Aspose.Zip, συμπεριλαμβανομένων των παραδοσιακών zip, δομών φακέλων zip και σύγχρονων μορφών όπως XAR και WIM.

## Τι σημαίνει “αποσυμπίεση πολλαπλών αρχείων”;
Η αποσυμπίεση πολλαπλών αρχείων σημαίνει την εξαγωγή κάθε καταχώρησης που βρίσκεται μέσα σε ένα αρχείο (ZIP, TAR κ.λπ.) και, προαιρετικά, τη γραφή κάθε αρχείου σε έναν προορισμό. Η λειτουργία αυτή είναι συχνή όταν λαμβάνετε πακεταρισμένα δεδομένα—αρχεία καταγραφής, εικόνες ή σύνολα ρυθμίσεων—που πρέπει να αποσυμπιεστούν πριν την επεξεργασία.

## Γιατί να χρησιμοποιήσετε το Aspose.Zip για .NET για αποσυμπίεση πολλαπλών αρχείων;
- **Performance‑optimized** εξαγωγή που διαχειρίζεται μεγάλα αρχεία χωρίς υπερβολική κατανάλωση μνήμης.  
- **Built‑in support** για κλασικό ZIP, σύγχρονες μορφές (XAR, WIM) και κρυπτογραφημένα αρχεία.  
- **Straightforward C# syntax** – δεν χρειάζεται να διαχειρίζεστε ρεύματα ή εξωτερικά εργαλεία (`aspose zip extract`).  
- **Cross‑platform** συμβατότητα, επιτρέποντας την εκτέλεση του ίδιου κώδικα σε Windows, Linux ή macOS.

## Συχνές Ερωτήσεις

**Q: Μπορώ να εξάγω ένα αρχείο zip απευθείας σε ρεύμα μνήμης;**  
A: Ναι, το Aspose.Zip σας επιτρέπει να διαβάσετε μια καταχώρηση σε ένα `MemoryStream` χωρίς να γράψετε στο δίσκο (`extract zip archive c#`).

**Q: Υποστηρίζει η βιβλιοθήκη εξαγωγή σε συγκεκριμένη δομή φακέλων;**  
A: Απόλυτα. Μπορείτε να καθορίσετε τον φάκελο εξόδου και το API θα επαναδημιουργήσει την εσωτερική ιεραρχία φακέλων του αρχείου.

**Q: Πώς εξάγω ένα zip αρχείο με προστασία κωδικού σε C#;**  
A: Παρέχετε τον κωδικό στη μέθοδο `Extract` (π.χ., `archive.Extract(outputPath, password)`).

**Q: Υπάρχει τρόπος να εμφανίσω τα περιεχόμενα του αρχείου χωρίς να τα εξάγω;**  
A: Ναι, μπορείτε να επαναλάβετε το `archive.Entries` για να ελέγξετε ονόματα αρχείων, μεγέθη και χρονικές σφραγίδες.

**Q: Τι γίνεται αν το αρχείο περιέχει διπλά ονόματα αρχείων;**  
A: Από προεπιλογή, η βιβλιοθήκη αντικαθιστά υπάρχοντα αρχεία· μπορείτε να αλλάξετε αυτή τη συμπεριφορά με την επιλογή `OverwriteMode`.

**Q: Μπορώ να εξάγω μόνο επιλεγμένες καταχωρήσεις από έναν φάκελο zip;**  
A: Ναι, φιλτράρετε το `archive.Entries` κατά όνομα ή επέκταση και καλέστε `Extract` στις επιλεγμένες καταχωρήσεις.

**Q: Πώς διαχειρίζεται το Aspose.Zip μεγάλα zip αρχεία σε συσκευές με περιορισμένη μνήμη;**  
A: Η βιβλιοθήκη χρησιμοποιεί lazy loading και streaming, έτσι μόνο η τρέχουσα καταχώρηση φορτώνεται στη μνήμη.
