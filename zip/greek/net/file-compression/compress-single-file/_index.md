## Βήμα 1: Ρύθμιση του Καταλόγου Εγγράφου

Ορίστε το φάκελο που περιέχει το αρχείο προέλευσης που θέλετε να συμπιέσετε. Αντικαταστήστε το placeholder με την πραγματική διαδρομή στο σύστημά σας.

```csharp
string dataDir = "Your Document Directory";
```

> **Συμβουλή:** Χρησιμοποιήστε `Path.Combine` για διαδρομές ανεξάρτητες από την πλατφόρμα, π.χ., `Path.Combine(dataDir, "alice29.txt")`.

## Βήμα 2: Δημιουργία Αρχείου Zip Χρησιμοποιώντας FileStream

Ανοίξτε ένα `FileStream` που δείχνει στο αρχείο εξόδου ZIP. Αυτό δείχνει την τεχνική **zip file using filestream**.

```csharp
using (FileStream zipFile = File.Open(dataDir + "CompressSingleFile_out.zip", FileMode.Create))
```

Η δήλωση `using` εγγυάται ότι η ροή κλείνει και το αρχείο εκκενώνεται σωστά.

## Βήμα 3: Προσθήκη Αρχείου στο Αρχείο

Τώρα ανοίξτε το αρχείο προέλευσης (`alice29.txt`) και προσθέστε το στο αρχείο. Αυτό είναι ο πυρήνας της λειτουργίας **c# compress file zip**.

```csharp
using (FileStream source1 = File.Open(dataDir + "alice29.txt", FileMode.Open, FileAccess.Read))
{
    using (var archive = new Archive(new ArchiveEntrySettings()))
    {
        archive.CreateEntry("alice29.txt", source1);

        // Save the archive
        archive.Save(zipFile);
    }
}
```

### Πώς λειτουργεί ο κώδικας
- **FileStream Setup** – Δημιουργεί σύνδεση με το αρχείο εξόδου ZIP.
- **CreateEntry** – Παίρνει το ρεύμα πηγής (`source1`) και το γράφει στο αρχείο υπό το όνομα `"alice29.txt"`.
- **Save** – Αποθηκεύει τα συμπιεσμένα δεδομένα στο `CompressSingleFile_out.zip`.

Μπορείτε να επαναλάβετε την κλήση `CreateEntry` για επιπλέον αρχεία, μετατρέποντας αυτό το απόσπασμα σε ένα πλήρες **zip archive tutorial c#**.

| **Πρόσβαση απορρίφθηκε** | **Ανεπαρκή δικαιώματα αρχείου** | Εκτελέστε το Visual Studio ως διαχειριστής ή δώστε δικαιώματα εγγραφής στο φάκελο |
| **Κενό αρχείο zip** | **`archive.Save` κλήθηκε εκτός του μπλοκ `using`** | Βεβαιωθείτε ότι το `archive.Save(zipFile);` βρίσκεται μέσα στο εσωτερικό μπλοκ `using` όπως φαίνεται |
