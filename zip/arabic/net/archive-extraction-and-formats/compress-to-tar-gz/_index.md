# إنشاء أرشيف tar وإضافة ملفات إلى tar باستخدام Aspose.Zip لـ .NET

## مقدمة

في تطبيقات .NET الحديثة، ** إنشاء أرشيف tar** و** إضافة ملفات إلى tar** بسرعة مطلوبية هو طلب شائع — سواء كنت تقوم بتجميع السجلات، أو إعداد البيانات لتخزين السحابي، أو بناء حزم النشر. يوفر Aspose.Zip لـ .NET واجهة برمجة تطبيقات خاصة بالأداء الجيد **لإضافة ملفات إلى tar**، ثم ضغط الأرشيف إلى تنسيق تنسيق **tar.gz** واسع الاستخدام. في هذا الدليل سنستعرض المراجعة الشاملة، من إعداد مشروعك لإنتاج ملف `archive.tar.gz` جاهز للإرسال.

## إجابات سريعة
- **ما المكتبة التي يجب استخدامها؟** Aspose.Zip for .NET
- **كيف يمكنني إضافة ملفات إلى tar؟** استخدم `TarArchive.CreateEntry` لكل ملف.
- **هل يمكنني الضغط مباشرة على tar.gz؟** نعم — استدعِ `SaveGzipped`.
- **هل أحتاج إلى ترخيص للإنتاج؟** يلزم وجود ترخيص Aspose للاستخدام غير التجريبي.
- **الإصدارات المدعومة من .NET؟** .NET Framework 4.5+، .NET Core 3.1+، .NET 5/6/7.

## ما هو "إضافة ملفات إلى القطران"؟

إضافة ملفات إلى أرشيف tar يعني تجميع ملفات متعددة في حاوية واحدة غير مضغوطة. تحافظ على بروتوكول tar على قاعدة الدليل وبيانات التعريف الخاصة بالملفات، مما يجعل مثاليًا للأرشفة قبل الضغط على الاختياري (مثل gzip) لإنشاء أرشيف **tar.gz**.

## لماذا نستخدم Aspose.Zip لضغط الملفات إلى tar.gz؟
- **لا أدوات خارجية** – كل شيء يعمل داخل كود .NET الخاص بك.
- **أداء عالي** – واجهة برمجة تطبيقات تعتمد على تنوعات الملفات الجديدة الجديدة.
- **منصات متعددة القطر** – تعمل على نظامي التشغيل Windows وLinux وmacOS دون تعديل.
- **مجموعة الميزات المتنوعة** – يدعم التشفير، حماية كلمة المرور، اتصالات العناوين المخصصة.

## كيفية إضافة ملفات إلى ملف tar باستخدام Aspose.Zip لـ .NET

### الخطوة 1: تحديد مجلد المستندات

أولاً، وجه الكود إلى المجلد الذي يحتوي على الملفات التي تريد أرشفتها.

```csharp
string dataDir = "Your Document Directory";
```

> **نصيحة احترافية:** استخدم `Path.Combine` عند بناء المسارات لتجنب مشاكل الفواصل الخاصة بالمنصة.

### الخطوة 2: إنشاء أرشيف TarGz

الآن سننشئ أرشيف tar، نضيف الإدخالات، ونضغطه في تدفق واحد سلس.

#### 2.1 تهيئة أرشيف Tar

```csharp
using (TarArchive archive = new TarArchive())
{
    // Entries will be added inside this block.
}
```

#### 2.2 إضافة الملفات - جوهر عملية "إضافة ملفات إلى ملف tar"

```csharp
archive.CreateEntry("alice29.txt", dataDir + "alice29.txt");
archive.CreateEntry("lcet10.txt", dataDir + "lcet10.txt");
```

كل استدعاء `CreateEntry` يأخذ **اسم الإدخال** (كيف سيظهر الملف داخل tar) و**مسار الملف المصدر** على القرص. يمكنك استدعاء `CreateEntry` بشكل متكرر لإضافة ملفات متعددة إلى tar في أرشيف واحد.

#### 2.3 الحفظ كملف Tar مضغوط (كيفية ضغط ملف tar.gz)

```csharp
archive.SaveGzipped(dataDir + "archive.tar.gz");
```

`SaveGzipped` يكتب محتويات tar إلى تدفق gzip، مما يمنحك ملف `archive.tar.gz` مضغوط جاهز للتوزيع.

## المشكلات والحلول الشائعة

- ** خطأ عدم العثور على الملف** – تأكد من أن `dataDir` ينتهي بالفاصل المناسب للمسار أو يستخدم `Path.Combine`.
- **الملفات الكبيرة ضغط غمرًا على الذاكرة** – استخدم التعديلات المعتمدة على فرانسيسكو (`CreateEntry` مع `Stream`) تحميل الملفات بالكامل في الذاكرة.
- **مخرجات gzip تالفة** – تأكد من إغلاق الأرشيف (`using` block) قبل الاتصال `SaveGzipped`.
