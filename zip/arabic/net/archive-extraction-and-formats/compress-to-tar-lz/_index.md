# كيفية ضغط ملفات متعددة بصيغة tar باستخدام Aspose.Zip لـ .NET

في تطوير .NET الحديث، يمكن أن يؤدي تجميع الملفات بكفاءة إلى تحسين كبير في حجم النشر وأوقات نقل الشبكة. **Compress multiple files tar** هو طلب شائع عندما تحتاج إلى أرشيف TAR مضغوط بخفة وزن باستخدام LZ للنسخ الاحتياطي أو التوزيع أو التحميل إلى السحابة. في هذا البرنامج التعليمي سنستعرض مثالًا واضحًا خطوة‑بخطوة **tar.lz compression example** باستخدام مكتبة Aspose.Zip، حتى تتمكن من إنشاء **tar.lz archive** بسرعة في تطبيقاتك الخاصة.

- **كم من الوقت تستغرق التنفيذ؟** حوالي 5‑10 دقائق لمثال أساسي.  
- **هل أحتاج إلى ترخيص؟** النسخة التجريبية المجانية تعمل للاختبار؛ يلزم ترخيص تجاري للإنتاج.

## كيفية ضغط ملفات متعددة بصيغة tar
يتناول هذا القسم مباشرة الكلمة المفتاحية الأساسية ويظهر لك الخطوات الدقيقة لـ **compress multiple files tar** باستخدام Aspose.Zip. النهج يعمل لأي عدد من الملفات، ويمكنك بسهولة تكييفه مع هياكل المجلدات الخاصة بك.

## ما هو ضغط tar.lz؟
`tar.lz` هو أرشيف TAR تم ضغطه باستخدام خوارزمية LZMA (المعروفة غالبًا ببساطة باسم **LZ**). يجمع بين بساطة تجميع الملفات في TAR ونسبة الضغط العالية لـ LZ، مما يجعله مثاليًا لملفات النسخ الاحتياطي، توزيع الحزم، أو أي سيناريو حيث يكون عرض النطاق الترددي مهمًا.

## لماذا تستخدم Aspose.Zip لـ .NET؟
Aspose.Zip يج abstracts التفاصيل منخفضة المستوى لإنشاء الأرشيفات، موفرًا لك API كائني نظيف. ستحصل على:

* **لا توجد تبعيات خارجية** – تنفيذ نقي بـ .NET.  
* **دعم متعدد المنصات** – يعمل على Windows وLinux وmacOS.  
* **ضغط LZ مدمج** – لا حاجة لتثبيت أدوات أصلية إضافية.  
* **معالجة أخطاء قوية** – الاستثناءات وصفية، مما يجعل تصحيح الأخطاء أسهل.

## استيراد مساحات الأسماء
أضف مساحات الأسماء المطلوبة حتى يعرف المترجم أين يجد الفئات التي سنستخدمها.

```csharp
using System;
using Aspose.Zip.Tar;
```

## كيفية إنشاء أرشيف tar.lz – دليل خطوة بخطوة

### الخطوة 1: ضغط ملف واحد
المثال الأول يوضح أبسط سيناريو – إضافة ملف واحد إلى أرشيف TAR ثم حفظه كملف **tar.lz**.

```csharp
//ExStart: CompressSingleFile
using (TarArchive archive = new TarArchive())
{
    archive.CreateEntry("alice29.txt", dataDir + "alice29.txt");
    archive.SaveLzipped(dataDir + "archive.tar.lz");
}
```

**شرح**

- `new TarArchive()` ينشئ حاوية TAR فارغة.  
- `CreateEntry` يضيف الملف `alice29.txt` من `dataDir` الخاص بك.  
- `SaveLzipped` يكتب الأرشيف إلى القرص ويطبق ضغط LZ، منتجًا `archive.tar.lz`.

### الخطوة 2: ضغط ملفات متعددة في أرشيف واحد
غالبًا ما تحتاج إلى تجميع عدة ملفات معًا. ما عليك سوى استدعاء `CreateEntry` لكل ملف قبل الحفظ. هذا يوضح **add files to tar lz** و**compress multiple files tar** بفعالية.

```csharp
//ExStart: CompressMultipleFiles
using (TarArchive archive = new TarArchive())
{
    archive.CreateEntry("alice29.txt", dataDir + "alice29.txt");
    archive.CreateEntry("lcet10.txt", dataDir + "lcet10.txt");
    archive.SaveLzipped(dataDir + "archive.tar.lz");
}
```

**شرح**

- الكود يتبع نفس النمط كما في الخطوة 1، لكنه يضيف إدخالًا ثانيًا (`lcet10.txt`).  
- يمكنك تكرار `CreateEntry` بقدر ما تحتاج؛ المكتبة تتعامل مع بنية TAR الداخلية تلقائيًا.

### الخطوة 3: تحديد دليل المستندات الخاص بك
استبدل العنصر النائب بالمسار الفعلي حيث توجد ملفات المصدر الخاصة بك. يُستخدم هذا المسار في الأمثلة أعلاه.

```csharp
string dataDir = "Your Document Directory";
```

**شرح**

- عيّن `dataDir` إلى مسار مجلد مؤهل بالكامل، مثال: `@"C:\\MyFiles\\"`.  
- الحفاظ على الدليل في متغير يجعل الكود قابلًا لإعادة الاستخدام وأسهل صيانة.
