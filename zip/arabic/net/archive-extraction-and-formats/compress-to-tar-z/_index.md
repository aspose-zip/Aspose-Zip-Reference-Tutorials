# إضافة ملفات إلى tar وضغطها إلى TarZ باستخدام Aspose.Zip لـ .NET

## المقدمة

إذا كنت بحاجة إلى **إضافة ملفات إلى tar** ثم ضغط الأرشيف إلى تنسيق TarZ، فإن Aspose.Zip لـ .NET يجعل العملية بأكملها سهلة وسلسة. في هذا البرنامج التعليمي سنستعرض كل خطوة — من إعداد المشروع إلى إنشاء أرشيف tar، إضافة الملفات، وأخيرًا حفظ ملف .tar.z مضغوط. في النهاية ستحصل على مقتطف يمكن إعادة استخدامه وإدراجه في أي تطبيق .NET.

## إجابات سريعة
- **ما المكتبة التي تتعامل مع إنشاء tar؟** Aspose.Zip لـ .NET  
- **كم عدد أسطر الكود؟** حوالي 15 سطرًا (بدون التعليقات)  
- **هل أحتاج إلى ترخيص للاختبار؟** يتوفر إصدار تجريبي مجاني؛ الترخيص مطلوب للإنتاج.  
- **الإصدارات المدعومة من .NET؟** .NET Framework 4.5+، .NET Core 3.1+، .NET 5/6+  
- **هل يمكنني ضغط المجلدات وليس الملفات فقط؟** نعم – يمكنك إضافة أدلة كاملة باستخدام حلقة.

## ما هو **add files to tar**؟

إضافة ملفات إلى أرشيف tar تقوم بدمجها في حاوية واحدة غير مضغومة تحافظ على بنية الدليل وبيانات تعريف الملفات. Tar هو تنسيق يونكس كلاسيكي ويُعد أساسًا للعديد من سير عمل الضغط، بما في ذلك تنسيق TarZ المستخدم في هذا الدليل.

## لماذا إضافة ملفات إلى tar قبل ضغطها إلى TarZ؟

- **قابلية النقل** – يعمل أرشيف tar عبر المنصات دون الحاجة للقلق بشأن معالجة كل ملف على حدة.  
- **السرعة** – إنشاء حاوية tar سريع؛ ثم يركز ضغط Z فقط على تقليل الحجم.  
- **التوافق** – العديد من الأدوات القديمة تتوقع وجود `.tar` قبل تطبيق ضغط يشبه gzip، وهو بالضبط ما يوفره `.tar.z`.

- مجلد على جهازك يحتوي على الملفات التي تريد أرشفتها. استبدل مسار العنصر النائب بالمسار الفعلي للمجلد الخاص بك.

## استيراد المساحات الاسمية

أضف بيانات `using` المطلوبة في أعلى ملف C# الخاص بك:

```csharp
using System;
using Aspose.Zip.Tar;
```

## دليل خطوة بخطوة

### الخطوة 1: تعريف دليل المستند الخاص بك

```csharp
string dataDir = "Your Document Directory";
```

> **نصيحة احترافية:** استخدم `Path.Combine` إذا كنت بحاجة إلى بناء المسارات بشكل ديناميكي؛ فهو يتجنب فقدان فواصل المسار على أنظمة تشغيل مختلفة.

### الخطوة 2: إنشاء أرشيف Tar وإضافة ملفات

#### 2.1: إنشاء مثيل أرشيف Tar

```csharp
using (TarArchive archive = new TarArchive())
{
    // Your code for creating the Tar archive goes here
}
```

#### 2.2: إضافة ملفات إلى الأرشيف  

داخل كتلة `using`، أضف كل ملف تريد تضمينه:

```csharp
archive.CreateEntry("alice29.txt", dataDir + "alice29.txt");
archive.CreateEntry("lcet10.txt", dataDir + "lcet10.txt");
```

يمكنك تكرار `CreateEntry` لعدد الملفات المطلوب، أو تنفيذ حلقة عبر دليل لإضافتها برمجيًا.

#### 2.3: حفظ ملف TarZ المضغوط  

بعد إضافة جميع الإدخالات، اضغط أرشيف tar إلى تنسيق `.tar.z`:

```csharp
archive.SaveZCompressed(dataDir + "archive.tar.z");
```

سيتم حفظ ملف `archive.tar.z` الناتج في نفس المجلد الذي حددته في `dataDir`.
