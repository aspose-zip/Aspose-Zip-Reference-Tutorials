## كيفية استخراج كلمة مرور ملف ZIP (فك ضغط ZIP محمي بكلمة مرور)

### الخطوة 1: إعداد مشروعك

أنشئ مشروعًا جديدًا في Visual Studio (وحدة تحكم أو سطح مكتب). أضف مرجعًا إلى ملف Aspose.Zip DLL الذي قمت بتحميله، وانسخ ملف ZIP المشفر إلى مجلد الإخراج الخاص بالمشروع (مثل `bin\Debug\net6.0`).

### الخطوة 2: تهيئة المتغيرات

عرّف الدليل الذي يحتوي على ملفات الاختبار. هذا يجعل الكود نظيفًا وقابلًا لإعادة الاستخدام:

```csharp
string dataDir = "YourDocumentDirectory";
```

### الخطوة 3: فك ضغط ملف مشفر بـ AES

الآن نصل إلى المنطق الأساسي الذي **يستخرج كلمة مرور ملف zip** ويقرأ المدخل المشفر. المقتطف أدناه يفتح الأرشيف، يزود كلمة المرور (`p@s$` في هذا المثال)، ويكتب المحتوى المستخرج إلى ملف جديد.

```csharp
//ExStart: DecompressAESEncryptedFile
using (FileStream fs = File.OpenRead(dataDir + "PasswordProtectWithAES256_out.zip"))
{
    using (var extracted = File.Create(dataDir + "alice_aesextracted_out.txt"))
    {
        using (Archive archive = new Archive(fs))
        {
            using (var decompressed = archive.Entries[0].Open("p@s$"))
            {
                byte[] b = new byte[8192];
                int bytesRead;
                while (0 < (bytesRead = decompressed.Read(b, 0, b.Length)))
                {
                    extracted.Write(b, 0, bytesRead);
                }
            }
        }
    }
}
//ExEnd: DecompressAESEncryptedFile
```

> **نصيحة احترافية:** إذا كنت بحاجة لاستخراج عدة مدخلات، قم بالتكرار عبر `archive.Entries` واستدعِ `Open(password)` لكل واحدة.

## كيفية فك ضغط ملفات ZIP المشفرة بـ AES

فئة `Archive` نفسها تعمل مع أي أرشيف مشفر بـ AES، بغض النظر عن طول المفتاح (128، 192 أو 256 بت). فقط قدم كلمة المرور الصحيحة، وستتعامل المكتبة مع فك التشفير داخليًا.
