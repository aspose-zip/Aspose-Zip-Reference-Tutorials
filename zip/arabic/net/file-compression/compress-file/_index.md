## كيفية ضغط الملفات c# باستخدام Aspose.Zip

قبل أن نغوص في الكود، دعنا نوضح لماذا قد تختار هذه المكتبة بدلاً من فئات `System.IO.Compression` المدمجة:

- **Rich API** – تدعم Cpio، Tar، Zip، GZip والمزيد.  
- **Pure .NET** – لا توجد ملفات DLL أصلية، مما يجعل النشر على Windows أو Linux أو macOS سهلًا.

- **Password support** – رغم أن Cpio لا يشفر، تتيح لك المكتبة إنشاء **zip archive password c#** عندما تنتقل إلى `ZipArchive`.

## ما هو ضغط الملفات ولماذا هو مهم؟

الآن، لنقسم مثال الكود إلى خطوات متعددة.

## الخطوة 1: تعيين دليل المستندات الخاص بك

قبل ضغط الملفات، عيّن الدليل الذي تُخزن فيه مستنداتك. استبدل `"Your Document Directory"` بالمسار الفعلي لدليل المستندات الخاص بك.

```csharp
string dataDir = "Your Document Directory";
```

## الخطوة 2: ضغط ملف

الآن، لننتقل إلى الكود الخاص بضغط ملف. يوضح هذا المثال كيفية ضغط الملفات باستخدام فئة `CpioArchive`.

```csharp
//ExStart: CompressFile
using (CpioArchive archive = new CpioArchive())
{
    archive.CreateEntries(dataDir);
    archive.Save(dataDir + "archive.cpio");
}
//ExEnd: CompressFile
Console.WriteLine("Successfully Compressed Files");
```

### الشرح

- **فئة `CpioArchive`** – تمثل أرشيف Cpio وتوفر طرقًا لإنشاء وتعديل مدخلات الأرشيف.  
- **طريقة `CreateEntries`** – تفحص الدليل المحدد وتضيف كل ملف إلى الأرشيف (مثالية لـ *c# file compression* للمجلدات الكاملة).  
- **طريقة `Save`** – تكتب الأرشيف إلى القرص؛ في هذا المثال يُحفظ الملف باسم `archive.cpio`.  
- **رسالة النجاح** – تؤكد أن عملية الضغط انتهت بدون أخطاء.
