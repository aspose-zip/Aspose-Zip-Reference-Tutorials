## 常見問題

**Q: 我可以在 .NET Core 上使用 Aspose.Zip 建立 zip archive .net 嗎？**  
A: 當然可以。Aspose.Zip 完全支援 .NET Core 3.1 及以上版本，同時支援 .NET 5/6/7。

**Q: 如何在不先解壓的情況下修改現有的壓縮檔？**  
A: 使用 “modify zip file c#” API——您可以直接在壓縮檔串流上新增、刪除或重新命名條目。

**Q: 哪種壓縮方法對文字檔案的壓縮率最高？**  
A: 一般而言，Bzip2 或 LZMA 對文字檔提供較高的壓縮率，但速度較 Deflate 慢。請兩者皆測試以找到最佳平衡。

**Q: 在建立 zip 壓縮檔時有沒有辦法加密檔案？**  
A: 有，Aspose.Zip 支援 AES 加密。您可在儲存前於 ZipArchive 物件設定密碼。

**Q: 我需要擔心檔名編碼問題嗎？**  
A: Aspose.Zip 會自動處理 Unicode 檔名，但若需與舊系統相容，您亦可自行設定編碼。
