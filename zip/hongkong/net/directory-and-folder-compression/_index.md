### 如何壓縮資料夾

1. **Instantiate the `ZipPackage` object** – 這代表您即將建立的壓縮檔。  
2. **Add the target directory** 使用 `AddFolder` 方法，會自動包含子資料夾與檔案。  
3. **Save the archive** 至您指定的位置，副檔名為 `.zip`。

## 常見問題與技巧
- **Large files:** 處理大於 2 GB 的檔案時，請啟用 **Zip64** 模式以避免大小限制。  
- **Path length issues:** 若資料夾層級超過 Windows 260 字元限制，請使用 `UseLongFileNames` 屬性。  
- **Performance tip:** 將 `CompressionLevel` 設為 `Fast` 以加速建置，或在需要最小檔案時設為 `Maximum`。

## 常見問題

**Q: 我可以使用 Aspose.Zip 建立受密碼保護的 zip 檔案嗎？**  
A: 可以。儲存壓縮檔時，您可以提供 `ZipPassword`，並選擇如 AES‑256 等加密演算法。

**Q: Aspose.Zip 是否支援串流大型檔案而不需將其完整載入記憶體？**  
A: 當然。您可以使用 `FileStream` 物件，讓您能有效率地壓縮或解壓任意大小的檔案。

**Q: 若需將大型壓縮檔分割成多個部分該怎麼辦？**  
A: 使用 `SplitArchive` 方法定義最大分割大小；Aspose.Zip 會自動產生連續的分割檔案。

**Q: 能否向現有的 zip 檔案新增檔案？**  
A: 可以。以 `Update` 模式開啟壓縮檔，然後呼叫 `AddFile` 或 `AddFolder` 以加入新內容。

**Q: 官方支援哪些 .NET 執行環境？**  
A: Aspose.Zip for .NET 支援 .NET Framework 4.5+、.NET Core 3.1+ 以及 .NET 5/6/7+。
