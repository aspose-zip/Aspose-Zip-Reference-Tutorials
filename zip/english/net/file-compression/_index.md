## Frequently Asked Questions

**Q: Can I use Aspose.Zip to both compress and extract archives in the same application?**  
A: Yes, the API provides methods for creating, updating, and extracting ZIP files, allowing you to handle the full archive lifecycle.

**Q: How do I control the compression level for a specific file?**  
A: Set the `CompressionLevel` property on the `ZipEntry` before adding the file to the archive. Higher levels yield smaller files but take longer to process.

**Q: Is it possible to add files to an existing ZIP without recompressing the whole archive?**  
A: Absolutely. Use the `Update` mode to append new entries, which preserves existing data and speeds up the operation.

**Q: Does Aspose.Zip support password‑protected archives?**  
A: Yes, you can specify a password when creating or extracting an archive to secure its contents.

**Q: What .NET versions are compatible with Aspose.Zip?**  
A: The library works with .NET Framework 4.5+, .NET Core 3.1+, .NET 5, .NET 6, and later versions.
