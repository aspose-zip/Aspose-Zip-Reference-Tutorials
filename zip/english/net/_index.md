## Quick Answers
- **What does “create zip archive .net” mean?** It’s the process of generating a ZIP file programmatically using .NET libraries such as Aspose.Zip.  
- **Which compression methods are available?** Bzip2, LZMA, PPMd, Deflate, and Store.

## Frequently Asked Questions

**Q: Can I create a ZIP archive without loading the entire file into memory?**  
A: Yes. Aspose.Zip supports streaming APIs that let you write entries directly to a file or response stream, which is ideal for large files.

**Q: How do I set a password for a ZIP archive?**  
A: Use the `ZipArchiveEntry.Password` property (or the `ZipArchive.Password` for the whole archive) and optionally specify the encryption type (e.g., AES‑256).

**Q: Is it possible to extract only specific files from a large archive?**  
A: Absolutely. You can enumerate entries and extract only the ones you need, avoiding unnecessary I/O.

**Q: Does Aspose.Zip handle Unicode file names?**  
A: Yes. The library fully supports UTF‑8 file names, ensuring compatibility across different operating systems.

**Q: What licensing options are available for production use?**  
A: Aspose offers both perpetual and subscription licenses. A free evaluation license is also provided for testing.
