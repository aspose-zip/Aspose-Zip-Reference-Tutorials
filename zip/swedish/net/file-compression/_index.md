## Vanliga frågor

**Q:** Kan jag använda Aspose.Zip för att både komprimera och extrahera arkiv i samma applikation?  
**A:** Ja, API‑et tillhandahåller metoder för att skapa, uppdatera och extrahera ZIP‑filer, vilket gör att du kan hantera hela arkivlivscykeln.

**Q:** Hur styr jag komprimeringsnivån för en specifik fil?  
**A:** Ställ in egenskapen `CompressionLevel` på `ZipEntry` innan du lägger till filen i arkivet. Högre nivåer ger mindre filer men tar längre tid att bearbeta.

**Q:** Är det möjligt att lägga till filer i ett befintligt ZIP utan att komprimera om hela arkivet?  
**A:** Absolut. Använd `Update`‑läget för att lägga till nya poster, vilket bevarar befintliga data och snabbar upp operationen.

**Q:** Stöder Aspose.Zip lösenordsskyddade arkiv?  
**A:** Ja, du kan ange ett lösenord när du skapar eller extraherar ett arkiv för att säkra dess innehåll.

**Q:** Vilka .NET‑versioner är kompatibla med Aspose.Zip?  
**A:** Biblioteket fungerar med .NET Framework 4.5+, .NET Core 3.1+, .NET 5, .NET 6 och senare versioner.
