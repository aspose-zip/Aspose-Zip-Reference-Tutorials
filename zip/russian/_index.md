## Как извлечь zip‑файлы с Aspose.Zip для .NET
Извлечение zip‑архива так же просто, как создать экземпляр класса `ZipFile` и вызвать его метод `ExtractAll`. API автоматически определяет структуру папок, обрабатывает перезапись файлов и учитывает любую защиту паролем, применённую к архиву.

### Обработка zip‑файлов, защищённых паролем
Если архив защищён паролем, передайте строку пароля в метод `ExtractAll`. Aspose.Zip расшифрует содержимое «на лету», позволяя работать с файлами так, как будто они не защищены.

### Шифрование zip‑архива при извлечении (повторное шифрование)
В сценариях, когда необходимо извлечь zip‑файл и сразу же повторно зашифровать его содержимое (например, при перемещении данных между защищёнными зонами), вы можете объединить извлечение с вспомогательным методом `CreateEncryptedArchive`. Такой подход гарантирует, что данные никогда не находятся на диске в незашифрованном виде.

### Сжатие нескольких файлов — Краткое резюме
Хотя данное руководство сосредоточено на извлечении, помните, что Aspose.Zip также отлично справляется с **compress files .net**. Вы можете добавить множество файлов в один архив одним вызовом, указать уровни сжатия и даже разбить большие архивы на тома.

## Распространённые проблемы и решения
- **Extraction fails with “Invalid password”** – Убедитесь, что указанный пароль совпадает с тем, который использовался при сжатии; пароли чувствительны к регистру.  
- **Large archives cause OutOfMemoryException** – Используйте потоковый API (`ExtractToStream`) для последовательной обработки файлов вместо загрузки всего архива в память.  
- **File name collisions** – Установите флаг `OverwriteExistingFiles`, чтобы контролировать, заменять ли существующие файлы или переименовывать их.

## Часто задаваемые вопросы

**Q: Can I extract a zip file without knowing its password?**  
A: Нет, Aspose.Zip требует правильный пароль для расшифровки защищённого паролем архива. Вы можете перехватить `InvalidPasswordException`, чтобы корректно обрабатывать неверные пароли.

**Q: Does Aspose.Zip support other archive formats like RAR or 7z?**  
A: Прямая поддержка ограничена ZIP, но вы можете комбинировать Aspose.Zip со сторонними библиотеками для этих форматов или воспользоваться учебником «Archive Extraction and Formats» для получения рекомендаций.

**Q: How do I extract only specific files from a large archive?**  
A: Используйте метод `ExtractEntry` для выбора отдельных записей по имени, избегая необходимости извлекать весь архив.

**Q: Is there a way to monitor extraction progress?**  
A: Да — подпишитесь на событие `ProgressChanged` объекта `ZipFile`, чтобы получать обновления в реальном времени.

**Q: What licensing is required for commercial use?**  
A: Для продакшн‑развёртываний требуется платная лицензия Aspose.Zip; бесплатная оценочная лицензия доступна для тестирования.

## Дополнительные советы и лучшие практики
- **Pro tip:** При работе с очень большими zip‑файлами предпочтительно использовать метод `ExtractToStream`, чтобы снизить потребление памяти.  
- **Tip:** Всегда проверяйте целостность архива с помощью `ValidateArchive` перед извлечением, чтобы раннее обнаружить повреждённые файлы.  
- **Warning:** Никогда не храните пароли в открытом виде; используйте безопасные провайдеры конфигураций или Azure Key Vault.
