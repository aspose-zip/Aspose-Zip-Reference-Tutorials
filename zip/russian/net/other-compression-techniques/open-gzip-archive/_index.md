## Шаг 2: Открыть GZip архив (Extract gzip file C#)

```csharp
//ExStart: OpenGZipArchive
//Extracts the archive and copies extracted content to file stream.
using (var archive = new GzipArchive(dataDir + "archive.gz"))
{
    using (var extracted = File.Create(dataDir + "data.bin"))
    {
        var unpacked = archive.Open();
        byte[] b = new byte[8192];
        int bytesRead;
        while (0 < (bytesRead = unpacked.Read(b, 0, b.Length)))
            extracted.Write(b, 0, bytesRead);
    }
}
//ExEnd: OpenGZipArchive
Console.WriteLine("Successfully Opened GZip Archive");
```

Этот код демонстрирует, как **extract a gzip file in C#** с помощью Aspose.Zip. Архив открывается, его содержимое передаётся потоково, а результат записывается в `data.bin`.

## Советы и лучшие практики

- **Используйте `Path.Combine`** чтобы избежать отсутствия разделителей пути при построении путей к файлам.  
- **Обрабатывайте потоки кусками** (как показано), чтобы снизить использование памяти, даже при работе с многогигабайтными файлами.  
- **Проверяйте архив** перед извлечением, проверяя `archive.IsValid`, если требуется дополнительная безопасность.  
- **Ведите журнал операции** чтобы можно было аудировать когда и как файлы сжаты или распакованы.
