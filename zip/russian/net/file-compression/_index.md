## Часто задаваемые вопросы

**Q: Можно ли использовать Aspose.Zip для создания zip archive .net на .NET Core?**  
A: Абсолютно. Aspose.Zip полностью поддерживает .NET Core 3.1 и более новые версии, а также .NET 5/6/7.

**Q: Как изменить существующий архив, не извлекая его сначала?**  
A: Используйте API “modify zip file c#” — вы можете добавлять, удалять или переименовывать записи напрямую в потоке архива.

**Q: Какой метод сжатия дает лучший коэффициент для текстовых файлов?**  
A: Как правило, Bzip2 или LZMA обеспечивают более высокое сжатие текста, но работают медленнее, чем Deflate. Протестируйте оба метода, чтобы найти оптимальный баланс.

**Q: Есть ли способ зашифровать файлы при создании zip‑архива?**  
A: Да, Aspose.Zip поддерживает шифрование AES. Вы можете установить пароль для объекта ZipArchive перед сохранением.

**Q: Нужно ли беспокоиться о кодировке имён файлов?**  
A: Aspose.Zip автоматически обрабатывает Unicode‑имена файлов, но при необходимости вы можете явно задать кодировку для совместимости со старыми системами.
