- **Какова основная цель Aspose.Zip?** Создавать, изменять и извлекать ZIP‑архивы из .NET приложений.

## Frequently Asked Questions

**Q: Можно ли использовать Aspose.Zip одновременно для сжатия и извлечения архивов в одном приложении?**  
A: Да, API предоставляет методы для создания, обновления и извлечения ZIP‑файлов, позволяя управлять полным жизненным циклом архива.

**Q: Как контролировать уровень сжатия для конкретного файла?**  
A: Установите свойство `CompressionLevel` у `ZipEntry` перед добавлением файла в архив. Более высокие уровни дают меньший размер, но требуют больше времени обработки.

**Q: Можно ли добавить файлы в существующий ZIP без повторного сжатия всего архива?**  
A: Абсолютно. Используйте режим `Update` для добавления новых записей, что сохраняет существующие данные и ускоряет операцию.

**Q: Поддерживает ли Aspose.Zip архивы, защищённые паролем?**  
A: Да, вы можете указать пароль при создании или извлечении архива для защиты его содержимого.

**Q: Какие версии .NET совместимы с Aspose.Zip?**  
A: Библиотека работает с .NET Framework 4.5+, .NET Core 3.1+, .NET 5, .NET 6 и более поздними версиями.
