Перед сжатием файлов укажите каталог, где хранятся ваши документы. Замените `"Your Document Directory"` реальным путём к вашему каталогу документов.

```csharp
string dataDir = "Your Document Directory";
```

## Шаг 2: Сжатие файла

Теперь перейдём к коду для сжатия файла. Этот пример демонстрирует, как сжимать файлы с помощью класса CpioArchive.

```csharp
//ExStart: CompressFile
using (CpioArchive archive = new CpioArchive())
{
    archive.CreateEntries(dataDir);
    archive.Save(dataDir + "archive.cpio");
}
//ExEnd: CompressFile
Console.WriteLine("Successfully Compressed Files");
```

### Объяснение

- **`CpioArchive` Class** – представляет Cpio‑архив и предоставляет методы для создания и управления записями архива.  
- **`CreateEntries` Method** – сканирует указанный каталог и добавляет каждый файл в архив (идеально подходит для *c# file compression* целых папок).  
- **`Save` Method** – записывает архив на диск; в этом примере файл сохраняется как `archive.cpio`.  
- **Success Message** – подтверждает, что операция сжатия завершилась без ошибок.

## Распространённые проблемы и решения

| Проблема | Причина | Решение |
|----------|---------|---------|
| **Empty archive** | `dataDir` указывает на неправильную папку или не содержит файлов. | Проверьте путь и убедитесь, что файлы существуют перед вызовом `CreateEntries`. |
| **Access denied** | Приложению не хватает прав для чтения исходных файлов или записи архива. | Запустите приложение с необходимыми привилегиями или скорректируйте ACL папки. |
| **Large files cause OutOfMemory** | Загрузка очень больших файлов в память одновременно. | Обрабатывайте файлы потоками или разбейте архив на несколько частей. |
