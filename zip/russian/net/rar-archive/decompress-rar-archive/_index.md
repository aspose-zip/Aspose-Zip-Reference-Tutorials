# Как извлечь RAR-архивы с помощью Aspose.Zip для .NET

## Введение

В разработке на .NET знание **как извлечь RAR** архивы быстро может сэкономить ваше время и упростить работу с файлами. Aspose.Zip для .NET предлагает простой API для **распаковки RAR-архивов**, **извлечения RAR в папку** и даже **чтения RAR‑файла в стиле C#**. Это руководство проведёт вас через каждый шаг, от настройки среды до извлечения файлов на диск.

## Быстрые ответы
- **Какая библиотека поддерживает извлечение RAR в .NET?** Aspose.Zip for .NET.  
- **Сколько строк кода требуется для извлечения RAR‑архива?** Около 10 строк, включая настройку.  
- **Можно ли извлечь RAR в определённую папку?** Да, используйте `ExtractToDirectory`, чтобы указать папку вывода.  
- **Требуется ли лицензия для продакшн?** Необходима коммерческая лицензия; доступна бесплатная пробная версия.  
- **Какие версии .NET поддерживаются?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6.

**В:** Как прочитать RAR‑файл в C# без извлечения?  
**О:** Вы можете перечислять записи с помощью `RarArchive` и читать потоки напрямую, но полное извлечение рекомендуется в большинстве сценариев.
