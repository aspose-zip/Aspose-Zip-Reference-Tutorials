## 자주 묻는 질문

**Q: .NET Core에서 Aspose.Zip를 사용해 zip archive .net를 만들 수 있나요?**  
A: 물론입니다. Aspose.Zip는 .NET Core 3.1 이상 및 .NET 5/6/7을 완벽히 지원합니다.

**Q: 기존 아카이브를 먼저 추출하지 않고 어떻게 수정할 수 있나요?**  
A: “modify zip file c#” API를 사용하세요 – 아카이브 스트림에서 직접 항목을 추가, 삭제 또는 이름 변경할 수 있습니다.

**Q: 텍스트 파일에 가장 높은 압축 비율을 제공하는 압축 방식은 무엇인가요?**  
A: 일반적으로 Bzip2 또는 LZMA가 텍스트에 더 높은 압축률을 제공하지만 Deflate보다 느립니다. 두 방식을 모두 테스트해 적절한 균형을 찾으세요.

**Q: zip 아카이브를 만들면서 파일을 암호화할 수 있는 방법이 있나요?**  
A: 예, Aspose.Zip는 AES 암호화를 지원합니다. 저장하기 전에 ZipArchive 객체에 비밀번호를 설정할 수 있습니다.

**Q: 파일 이름 인코딩에 대해 신경 써야 하나요?**  
A: Aspose.Zip는 Unicode 파일명을 자동으로 처리하지만, 레거시 시스템과의 호환성이 필요하면 인코딩을 명시적으로 설정할 수 있습니다.
