# เพิ่มไฟล์ลงใน tar และบีบอัดเป็น TarBz2 ด้วย Aspose.Zip สำหรับสมบูรณ์ของเราเกี่ยวกับ **การเพิ่มไฟล์ลงใน tar** และบีบอัดเป็นรูปแบบ TarBz2 ด้วย Aspose.Zip สำหรับ .NET ไม่ว่าคุณจะกำลังสร้างยูทิลิตี้สำรองข้อมูล จัดเตรียมแพคเกจการปรับใช้ หรือเพียงต้องการไฟล์อาร์ไคฟ์ที่กะทัดรัดสำหรับการแจกจ่าย บทเรียนนี้จะพาคุณผ่านทุกขั้นตอนด้วยคำอธิบายที่ชัดเจน เคล็ดลับจากโลกจริง และสถานการณ์การใช้งานที่เป็นประโยชน์

ก่อนที่เราจะเริ่ม ให้แน่ใจว่าคุณมีทุกอย่างที่ต้องการแล้ว

## คำตอบสั้น
- **ควรใช้ไลบรารีอะไร?** Aspose.Zip สำหรับ .NET  
- **การทำงานนี้ใช้เวลานานแค่เซนส์หรือไม่?** จำเป็นต้องมีไลเซนบีบอัดหลายไฟล์ได้หรือไม่?** ได้ – เพิ่มรายการเท่าใดก็ได้ลงในอาร์ไคฟ์ Tar

## “add files to tar” คืออะไร?
การเพิ่มไฟล์ลงใน **tar** (Tape Archive) จะสร้างคอนเทนเนอร์ที่ไม่ได้บีบอัดซึ่งคงโครงสร้างไดเรกทอรีและเมตาดาต้าไฟล์ไว้ เมื่อคุณนำมาบีบอัดด้วย Bzip2 ผลลัพธ์จะเป็นอาร์ไคฟ์ **tar.bz2** (TarBz2) – เหมาะสำหรับการจัดเก็บและถ่ายโอนที่มีประสิทธิภาพ

## ทำไมต้องบีบอัดไฟล์เป็น TarBz2 ด้วย Aspose.Zip?
- **ความเร็วและความง่าย** – คำสั่ง API บรรทัดเดียวจัดการการสร้าง tar และการบีบอัด Bzip2 ทั้งหมด  
- **ข้ามแพลตฟอร์ม** – ทำงานบน Windows, Linux, และ macOS .NET runtime  
- **การควบคุมระดับละเอียด** – เลือกไฟล์ที่ต้องการรวม ตั้งชื่อรายการแบบกำหนดเอง และสตรีมโดยตรงไปยังดิสก์

> **เคล็ดลับมืออาชีพ:** เก็บไฟล์ต้นฉบับไว้ในโฟลเดอร์เฉพาะเพื่อหลีกเลี่ยงการรวมไฟล์ที่ไม่ต้องการโดยบังเอิญ

## นำเข้า Namespaces

ก่อนอื่นให้นำเข้า namespaces ที่จำเป็นเพื่อให้คุณเข้าถึงคลาส Tar และ Bzip2 ของ Aspose.Zip

```csharp
using System;
using System.ComponentModel;
using Aspose.Zip.Bzip2;
using Aspose.Zip.Tar;
```

## ขั้นตอนที่ 1: ตั้งค่าโฟลเดอร์เอกสาร

กำหนดพาธที่ชี้ไปยังโฟลเดอร์ที่บรรจุไฟล์ที่คุณต้องการอาร์ไคฟ์

```csharp
string dataDir = "Your Document Directory";
```

> แทนที่ `"Your Document Directory"` ด้วยพาธแบบเต็มหรือแบบสัมพันธ์ไปยังโฟลเดอร์ต้นฉบับของคุณ

## ขั้นตอนที่ 2: เพิ่มไฟล์ลงใน tar และสร้างอาร์ไคฟ์ TarBz2

หัวใจของกระบวนการคือการสร้าง `TarArchive` เพิ่มรายการ แล้วห่อหุ้มด้วย `Bzip2Archive` โค้ดด้านล่างแสดง **วิธีสร้าง tarbz2** ด้วยรูปแบบ disposable‑pattern ที่สะอาด

```csharp
//ExStart: CompressFile
using (Bzip2Archive bz2 = new Bzip2Archive())
{
    using (TarArchive archive = new TarArchive())
    {
        archive.CreateEntry("alice29.txt", dataDir + "alice29.txt");
        archive.CreateEntry("lcet10.txt", dataDir + "lcet10.txt");

        bz2.SetSource(archive);
        bz2.Save(dataDir + "archive.tar.bz2");
    }
}
```

- `CreateEntry` เพิ่มไฟล์แต่ละไฟล์ลงในคอนเทนเนอร์ **tar**  
- `bz2.SetSource(archive)` บอกให้ Bzip2 archive บีบอัดสตรีม tar ทั้งหมด  
- `bz2.Save(...)` เขียนไฟล์ **tar.bz2** สุดท้ายลงดิสก์

**เคล็ดลับ:** เพื่อ **บีบอัดไฟล์เป็น tarbz2** เป็นกลุ่ม ให้ทำซ้ำ `archive.CreateEntry` สำหรับทุกไฟล์ที่ต้องการ

## วิธีสร้างอาร์ไคฟ์ tarbz2 – รายการตรวจสอบทาง (`dataDir`)  
2. เริ่มต้น `TarArchive` และเพิ่มไฟล์แต่ละไฟล์ด้วย `CreateEntry`  
3. ห่อหุ้ม tar archive ด้วย `Bzip2Archive`  
4. เรียก `Save` เพื่อเขียนไฟล์ **tar.bz2**  

การทำตามรายการตรวจสอบนี้จะทำให้เวิร์กโฟลว์ **การสร้างอาร์ไคฟ์ tarbz2** สม่ำเสมอ

## ปัญหาที่พบบ่อย & วิธีแก้

| ปัญหา |ไฟล์ไม่พบ** | พาธ `dataDir` ไม่ถูกต้องหรือขาดส่วนขยายไฟล์ | ตรวจสอบพาธเต็มและยืนยันว่าไฟล์มีอยู่ |
| **อาร์ไคฟ์ว่าง** | ไม่มีรายการใดถูกเพิ่มก่อน `bz2.Save` | เพิ่มอย่างน้อยหนึ่งการเรียก `CreateEntry` |
| **สิทธิ์ถูกปฏิเสธ** | แอปไม่มีสิทธิ์เขียนในโฟลเดอร์ผลลัพธ์ | รันแอปด้วยสิทธิ์ที่เหมาะสมหรือเลือกโฟลเดอร์ที่เขียนได้ |
