# วิธีเก็บข้อมูล – คืนนี้ด้วย Aspose.Zip สำหรับ .NET

โดยทั่วไป **วิธีการ zip โฟลเดอร์** อย่างชัดในแอปพลิเคชัน .NET คุณมาถูกที่แล้วในบทแนะนำนี้เราจะต้องใช้ระบบทั้งหมด—เราจะ **บีบอัดไดเรกทอรีเป็น zip**, เราจะแสดงที่แน่นอนเพื่อ **แยก zip ไปยังไดเรกทอรี** (หรือดำเนินการวิธีการ unzip โฟลเดอร์) เมื่อทำเสร็จแล้วคุณจะมีรูปแบบโปรแกรมที่อีกครั้งกลับมาใช้การทำงานของโฟลเดอร์ zip ที่ทำงานบน .NET Framework, .NET Core, และ .NET5/6+.

## คำตอบด่วน
- ** “compress directory to zip” หมายความว่าอย่างไร** ฟังก์ชั่นการประมวลผลเป็นไฟล์.zipอัจฉริยะเพียงตัวเดียว
- **ฉันจะแตกไฟล์ zip ไปยังไดเร็กทอรีได้อย่างไร?** ใช้เมธอด `Archive.ExtractToDirectory` ตามคู่มือคู่มือ
- **รองรับ .NET เวอร์ชันใดบ้างที่รองรับ** .NET Framework, .NET Core, และ .NET5/6+ รุ่นที่ทันสมัยทั้งหมด
- **จำเป็นต้องมีใบอนุญาตสำหรับการผลิตหรือไม่** ต้องอาศัยลิขสิทธิ์ของ Aspose.Zip บนคอมพิวเตอร์ที่โหดร้าย
- **ฉันสามารถดำเนินการนี้โดยอัตโนมัติในไปป์ไลน์ CI/CD ได้หรือไม่** จะต้องเพิ่มโค้ดเดียวกันนี้ลงในการควบคุมของคุณ

## “วิธีการ zip โฟลเดอร์” คืออะไร?
**วิธีการ zip โฟลเดอร์** คุณสมบัติพิเศษสำหรับไฟล์และการย่อยทั้งหมดภายในมารวมกันเป็นไฟล์เดียวที่สามารถเลือกขนาดได้,...

## เหตุใดจึงต้องใช้ Aspose.Zip สำหรับ .NET
Aspose.Zip ให้ API **pure‑managed** ที่ไม่ต้องอาศัย DLL แบบเนทีฟ, รองรับไฟล์อาร์ไคส่วนใหญ่, และจัดการกรณีพิเศษเช่น **zip archive การป้องกันด้วยรหัสผ่าน** และชื่อไฟล์ Unicode สำหรับการรำลึกถึงการปรับให้ทำงานได้อย่างมีประสิทธิภาพเพื่อตรวจสอบเมื่อต้องใช้โฟลเดอร์ zip ด้วยโปรแกรมที่ต้องการรวบรวมข้อมูลจำนวนมาก

## ข้อกำหนดเบื้องต้น
- **Aspose.Zip สำหรับ .NET** ติดตั้งไลบรารี่แล้ว (ดาวน์โหลดได้จาก [ที่นี่](https://releases.aspose.com/zip/net/))
- และอื่น ๆ อีกมากมายที่สามารถพบได้ – กำหนดพาธของมันในนวนิยาย `dataDir`
- บางครั้งการพัฒนา .NET (Visual Studio, VSCode, หรือ IDE ที่คุณชอบ)

## นำเข้าเนมสเปซ
นำมาสเปซเซสเข้ามาโคป:

```csharp
using Aspose.Zip;
using System.IO;
```

## คำแนะนำทีละขั้นตอน

### ขั้นตอนที่ 1: บีบอัดไดเรกทอรีเป็น Zip (โฟลเดอร์ zip โดยทางโปรแกรม)
เราจะสร้างไฟล์ zip จากไดเรกทอรีที่คุณวางแผนจะทำการแตกไฟล์ในภายหลัง ตัวช่วย `CompressDirectory.Run()` จะทำหน้าที่หลัก

```csharp
string dataDir = "Your Document Directory";
CompressDirectory.Run();
```

> **เคล็ดลับสำหรับมือโปร:** หรือแปลว่า `CompressDirectory` จะบรรจุไฟล์ทุกไฟล์ใน `dataDir` ไฟล์ `CompressDirectory_out.zip` ไม่จำเป็นต้องสร้างไฟล์ผลลัพธ์ให้ระบบควบคุมตามความต้องการของคุณ

### ขั้นตอนที่ 2: ขยายขนาดโฟลเดอร์ – วิธีแตกไฟล์โฟลเดอร์ใน .NET

#### ขั้นตอนที่ 2.1: เปิดไฟล์ซิป
เปิดอาร์ไคฟ์ที่สร้างขึ้นด้วย `FileStream`. การทำเช่นนี้เตรียมไฟล์สำหรับการอ่าน

```csharp
using (FileStream zipFile = File.Open(dataDir + "CompressDirectory_out.zip", FileMode.Open))
{
```

#### ขั้นตอนที่ 2.2: สร้างอินสแตนซ์การเก็บถาวร
สร้างอินสแตนซ์ของอ็อบเจกต์ `Archive` ซึ่งเป็นตัวแทนของคอนเทนเนอร์ zip

```csharp
    using (var archive = new Archive(zipFile))
    {
```

#### ขั้นตอนที่ 2.3: แยกไปยังไดเรกทอรี
สุดท้าย ให้แตกเนื้อหาไปยังโฟลเดอร์ใหม่ นี่คือขั้นตอน **extract zip to directory**

```csharp
        archive.ExtractToDirectory(dataDir + "DecompressFolder_out");
    }
}
```

## ทำไมเรื่องนี้ถึงสำคัญ
- **ความสม่ำเสมอ:** ไลบรารีเดียวกันสำหรับความหิวและการแตกไฟล์รับประกันรูปแบบอาร์ไคร์ที่วิญญาณ
- **Performance:** Aspose.Zip สตรีมข้อมูลอย่างมีประสิทธิภาพแม้ไฟล์อาร์ไคฟ์หลายส่วนสามารถจัดการได้ในระดับที่ต่ำ
- **ความปลอดภัย:** การรักษาความปลอดภัยด้วยรหัสผ่านในบางครั้งอาจมีอาร์ไคฟ์ zip เขียนโค้ดเพิ่มเติม

## กรณีการใช้งานทั่วไป
- **การสำรองข้อมูลอัตโนมัติ** – ซอฟต์แวร์เก็บข้อมูลประจำคืนและซอฟต์แวร์ที่เก็บข้อมูลสตอเรจ
- **แพ็คเกจการปรับใช้งาน** – รวมไฟล์ทรัพยากรเว็บแบบสแตนด์อโลนก่อนทำการเผยแพร่ไปยังเซิร์ฟเวอร์
- **การแลกเปลี่ยนข้อมูล** – ส่งความร้อนของไฟล์ระหว่างบริการเป็นอาร์ไคฟ์เดียว

## ปัญหาและแนวทางแก้ไขทั่วไป
| นั่น | สาเหตุที่ทำให้ | วิธีแก้ |
|---------|--------------|-----|
| `UnauthorizedAccessException` เมื่อแยก | เป้าหมายสามารถอ่านได้อย่างเดียวหรือกำลังถูกใช้งาน | การตัดต่อพาธปลายทางสามารถเขียนได้ในส่วนของบันทึก |
| ล้างโฟลเดอร์เอาต์พุตหลังจากการแตกไฟล์ | พาธ zip เริ่มต้นทางผิดพลาด | ตัดอีกครั้งว่า `dataDir + "CompressDirectory_out.zip"` ชี้ไปที่ไฟล์ที่ถูกต้อง |
| ไฟล์ขนาดใหญ่ทำให้เกิด OutOfMemoryException | ใช้ขนาดบัฟเฟอร์เริ่มต้นกับอาร์ไคฟ์ขนาดใหญ่มาก | ใช้ `ArchiveOptions` เพื่อเพิ่มขนาดเบาะหรือสตรีมไฟล์เป็น |
