# สกัดไฟล์ zip ที่มีการป้องกันด้วยรหัสผ่านด้วย Aspose.Zip สำหรับ .NET

## บทนำ

ยินดีต้อนรับ! ในบทแนะนำที่ครอบคลุมนี้คุณจะได้เรียนรู้ **วิธีสกัดไฟล์ zip ที่มีการป้องกันด้วยรหัสผ่าน** ที่ใช้การเข้ารหัส AES ด้วย Aspose.Zip สำหรับ .NET ไม่ว่าคุณจะกำลังสร้างยูทิลิตี้บนเดสก์ท็อป, บริการบนคลาวด์ หรือ งานแบตช์อัตโนมัติ การสามารถ *ถอดรหัส zip ที่มีการป้องกันด้วยรหัสผ่าน* และ *แตกไฟล์ zip ที่ได้รับการป้องกัน* เป็นความต้องการที่พบบ่อย เราจะพาคุณผ่านทุกขั้นตอนที่คุณต้องการ — ตั้งแต่การติดตั้งไลบรารีจนถึงการสตรีมเนื้อหาที่ถอดรหัสไปยังดิสก์ — ด้วยโค้ด C# ที่สะอาดและง่ายต่อการทำตาม

## คำตอบสั้น
- **อะไรหมายถึง “สกัดไฟล์ zip ที่มีการป้องกันด้วยรหัสผ่าน”?** เป็นกระบวนการเปิดไฟล์ ZIP ที่มีการป้องกันด้วยรหัสผ่านและดึงข้อมูลภายในโดยโปรแกรม  
- **ไลบรารีใดที่จัดการการถอดรหัส AES?** Aspose.Zip สำหรับ .NET มีการสนับสนุน AES‑256 แบบเนทีฟโดยไม่ต้องพึ่งพาไลบรารีเพิ่มเติม  
- **ฉันต้องการไลเซนส์สำหรับการผลิตหรือไม่?** ใช่ – จำเป็นต้องมีไลเซนส์เชิงพาณิชย์สำหรับการผลิต; มีรุ่นทดลองฟรีสำหรับการประเมินผล  
- **ฉันสามารถใช้กับ .NET 6+ ได้หรือไม่?** แน่นอน – ไลบรารีนี้ตั้งเป้าหมายที่ .NET Standard 2.0 และทำงานกับ .NET 6, .NET 7, และรุ่นต่อ ๆ ไป  
- **ขั้นตอนโค้ดทั่วไปเป็นอย่างไร?** โหลดอาร์ไคฟ์ด้วยรหัสผ่าน, ค้นหาเอนทรี, และสตรีมไบต์ที่ถอดรหัสไปยังไฟล์  

## วิธีสกัดไฟล์ zip ที่มีการป้องกันด้วยรหัสผ่าน

ด้านล่างเป็นคำแนะนำแบบทีละขั้นตอนที่แสดงอย่างชัดเจนวิธีเปิดอาร์ไคฟ์ที่เข้ารหัส AES และเขียนเอนทรีที่ถอดรหัสลงดิสก์

### การดำเนินการ “เปิดอาร์ไคฟ์ที่เข้ารหัส” คืออะไร?

การเปิดอาร์ไคฟ์ที่เข้ารหัสหมายถึงการโหลดไฟล์ ZIP ที่ได้รับการป้องกันด้วยรหัสผ่าน (โดยค่าเริ่มต้นคือ AES‑256) แล้วอ่านเอนทรีของมันโดยไม่ต้องจัดการการเข้ารหัสด้วยตนเอง Aspose.Zip จะทำหน้าที่ซ่อนรายละเอียดระดับต่ำ ให้คุณมุ่งเน้นที่ตรรกะธุรกิจของคุณ

### ทำไมต้องใช้ Aspose.Zip สำหรับ C# เพื่อถอดรหัสไฟล์ ZIP แบบ AES?

- **รองรับ AES เต็มรูปแบบ** – จัดการคีย์ 128‑, 192‑ และ 256‑บิตโดยอัตโนมัติ.  
- **API ที่เรียบง่าย** – เพียงบรรทัดเดียวของโค้ดเพื่อระบุรหัสผ่าน (`DecryptionPassword`).  
- **ไม่มีการพึ่งพาภายนอก** – ไม่จำเป็นต้องบรรจุ OpenSSL หรือไลบรารีเนทีฟอื่น.  
- **การจัดการข้อผิดพลาดที่แข็งแรง** – ปล่อยข้อยกเว้นที่ชัดเจนสำหรับรหัสผ่านผิดหรืออาร์ไคฟ์เสียหาย.  

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะลงลึกในโค้ด, โปรดตรวจสอบว่าคุณมีข้อกำหนดต่อไปนี้พร้อมใช้งาน:

- Aspose.Zip สำหรับ .NET: ตรวจสอบว่าคุณได้ติดตั้งไลบรารี Aspose.Zip แล้ว คุณสามารถค้นหาเอกสารได้ [ที่นี่](https://reference.aspose.com/zip/net/).  
- ตัวอย่างไฟล์ที่เข้ารหัส AES: ดาวน์โหลดไฟล์ตัวอย่างที่เข้ารหัส AES จาก [ลิงก์นี้](https://releases.aspose.com/zip/net/).  
- โฟลเดอร์เอกสารของคุณ: ตั้งค่าโฟลเดอร์ที่คุณต้องการเก็บไฟล์ที่แตกออกมา แทนที่ “Your Document Directory” ในโค้ดสแนปด้วยเส้นทางโฟลเดอร์จริงของคุณ.  

## นำเข้า Namespaces

ในโค้ดสแนปที่ให้มา, คุณจะสังเกตเห็นการใช้ Namespaces ต่าง ๆ โปรดแน่ใจว่าได้รวมสิ่งเหล่านี้ในโปรเจคของคุณ:

```csharp
using System.IO;
using Aspose.Zip;
```

## ขั้นตอนที่ 1: กำหนดไดเรกทอรีทรัพยากร

ระบุเส้นทางไปยังโฟลเดอร์ที่มีไฟล์ ZIP ที่เข้ารหัสของคุณและที่ไฟล์ที่สกัดจะถูกเขียน.

```csharp
string dataDir = "Your Document Directory";
```

## ขั้นตอนที่ 2: เปิดอาร์ไคฟ์ที่เข้ารหัส

คอนสตรัคเตอร์ `Archive` รับอ็อบเจ็กต์ `ArchiveLoadOptions` ที่คุณสามารถตั้งค่า `DecryptionPassword` ได้ นี่คือแกนหลักของการดำเนินการ **decrypt zip password**.

```csharp
using (FileStream fs = File.OpenRead(dataDir + "StoreMutlipleFilesWithoutCompressionWithPassword_out.zip"))
{
    using (var extracted = File.Create(dataDir + "alice_aesextracted_out.txt"))
    {
        using (Archive archive = new Archive(fs, new ArchiveLoadOptions() { DecryptionPassword = "p@s$" }))
        {
            // Continue to the next steps...
        }
    }
}
```

## ขั้นตอนที่ 3: แตกเอนทรีที่เข้ารหัส

เมื่ออาร์ไคฟ์ถูกเปิดแล้ว, คุณสามารถอ่านเอนทรีแรก (หรือเอนทรีใดก็ได้ที่ต้องการ) และเขียนไบต์ที่ถอดรหัสไปยังไฟล์ผลลัพธ์ นี่เป็นการสาธิต **c# extract encrypted zip** ในรูปแบบสตรีมมิ่ง.

```csharp
using (var decompressed = archive.Entries[0].Open())
{
    byte[] b = new byte[8192];
    int bytesRead;
    while (0 < (bytesRead = decompressed.Read(b, 0, b.Length)))
    {
        extracted.Write(b, 0, bytesRead);
    }
}
```
