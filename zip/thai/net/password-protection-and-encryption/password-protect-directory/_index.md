# สร้างไฟล์ zip ที่มีการป้องกันด้วยรหัสผ่านสำหรับไดเรกทอรี .NET – บทเรียน Aspose.Zip

ในคู่มือนี้คุณจะ **สร้างไฟล์ zip ที่มีการป้องกันด้วยรหัสผ่าน** สำหรับไดเรกทอรีทั้งหมดโดยใช้ไลบรารี Aspose.Zip สำหรับ .NET ไม่ว่าคุณจะต้องการ **เข้ารหัสโฟลเดอร์**, ปกป้องไฟล์สำรอง, หรือเพียงแค่จำกัดการเข้าถึงข้อมูลที่สำคัญ, บทเรียนขั้นตอนต่อขั้นตอนนี้จะแสดงให้คุณเห็นวิธีทำด้วยโค้ด C# ที่สะอาดและเข้าใจง่าย

## คำตอบสั้น
- **แนะนำให้ใช้ไลบรารีอะไร?** Aspose.Zip สำหรับ .NET  
- **สามารถเข้ารหัสโฟลเดอร์ทั้งหมดได้หรือไม่?** ได้ – เพียงแค่ชี้ API ไปที่โฟลเดอร์ที่ต้องการบีบอัด  
- **การเปลี่ยนรหัสผ่านของ zip รองรับหรือไม่?** แน่นอน, ใช้ `TraditionalEncryptionSettings`  
- **ต้องมีลิขสิทธิ์สำหรับการใช้งานในโปรดักชันหรือไม่?** จำเป็นต้องมีลิขสิทธิ์ Aspose.Zip ที่ถูกต้องสำหรับการใช้งานเชิงพาณิชย์  
- **ทำงานกับ .NET Core/5/6 ได้หรือไม่?** ได้, API รองรับรันไทม์ .NET รุ่นใหม่ทั้งหมด  

## “สร้างไฟล์ zip ที่มีการป้องกันด้วยรหัสผ่าน” คืออะไร?
การสร้างไฟล์ zip ที่มีการป้องกันด้วยรหัสผ่านหมายถึงการบีบอัดไฟล์หรือไดเรกทอรีลงในไฟล์ ZIP พร้อมกับการเข้ารหัส เพื่อให้ไฟล์ที่บีบอัดสามารถเปิดได้เฉพาะเมื่อใส่รหัสผ่านที่ถูกต้องเท่านั้น วิธีนี้ช่วยปกป้องเนื้อหาจากการเข้าถึงโดยไม่ได้รับอนุญาต

## ทำไมต้องใช้ Aspose.Zip สำหรับการป้องกันไดเรกทอรีด้วยรหัสผ่านใน .NET?
Aspose.Zip มี API ที่เรียบง่ายและประสิทธิภาพสูง รองรับ **c# zip password protection**, การเข้ารหัสแบบ ZipCrypto แบบดั้งเดิม, และการเข้ารหัส AES มันจัดการกับไดเรกทอรีขนาดใหญ่ได้อย่างมีประสิทธิภาพและสามารถรวมเข้ากับโปรเจกต์ .NET ใดก็ได้อย่างราบรื่น

## สิ่งที่ต้องเตรียมก่อน
ก่อนเริ่มทำตามขั้นตอน, ตรวจสอบให้แน่ใจว่าคุณมี:

- ความรู้พื้นฐานด้านการเขียนโปรแกรม C#  
- Visual Studio (รุ่นใดก็ได้ที่อัปเดต)  
- ไลบรารี Aspose.Zip สำหรับ .NET – ดาวน์โหลดได้ **[ที่นี่](https://releases.aspose.com/zip/net/)**  
- โฟลเดอร์บนดิสก์ที่คุณต้องการป้องกันด้วยรหัสผ่าน  

## นำเข้า Namespaces
เพิ่ม Namespaces ที่จำเป็นลงในไฟล์ C# ของคุณเพื่อให้คอมไพเลอร์รู้ว่าจะหาคลาสของ Aspose.Zip จากที่ไหน

```csharp
using Aspose.Zip;
using System.IO;
using Aspose.Zip.Saving;
```

## ขั้นตอนที่ 1: ตั้งค่าพาธไปยังไดเรกทอรีทรัพยากร
กำหนดพาธที่ชี้ไปยังไดเรกทอรีที่คุณต้องการบีบอัดและป้องกัน

```csharp
string dataDir = "Your Document Directory";
```

## ขั้นตอนที่ 2: ป้องกันไดเรกทอรีด้วยรหัสผ่าน
ใช้ `TraditionalEncryptionSettings` เพื่อระบุรหัสผ่านและสร้างไฟล์ที่เข้ารหัส นี่คือหัวใจของ **c# zip password protection**

```csharp
//ExStart: PasswordProtectDirectory
using (FileStream zipFile = File.Open(dataDir + "all_corpus_encrypted_out.zip", FileMode.Create))
{
    DirectoryInfo corpus = new DirectoryInfo(".\\CanterburyCorpus");
    using (var archive = new Archive(new ArchiveEntrySettings(null, new TraditionalEncryptionSettings("p@s$"))))
    {
        archive.CreateEntries(corpus);
        archive.Save(zipFile);
        //ExEnd: PasswordProtectDirectory
    }
}
```

## ขั้นตอนที่ 3: คำอธิบายโค้ด
- **การสร้างไฟล์ผลลัพธ์:** `File.Open(..., FileMode.Create)` เปิด (หรือสร้าง) ไฟล์ ZIP ที่จะเก็บข้อมูลที่เข้ารหัส  
- **การเลือกโฟลเดอร์ต้นทาง:** `new DirectoryInfo(".\\CanterburyCorpus")` บอก Aspose.Zip ว่าให้บีบอัดไดเรกทอรีใด  
- **การตั้งรหัสผ่าน:** `new TraditionalEncryptionSettings("p@s$")` กำหนดรหัสผ่านที่จะป้องกันไฟล์ ZIP  
- **การเพิ่มรายการและบันทึก:** `archive.CreateEntries(corpus)` เพิ่มไฟล์ทุกไฟล์ในโฟลเดอร์, และ `archive.Save(zipFile)` เขียนไฟล์ ZIP ที่เข้ารหัสลงดิสก์  

## วิธีเปลี่ยนรหัสผ่านของ zip หลังจากสร้างแล้ว?
หากต้องการ **เปลี่ยนรหัสผ่านของ zip**, เพียงสร้างไฟล์ ZIP ใหม่อีกครั้งโดยใช้อินสแตนซ์ `TraditionalEncryptionSettings` ที่มีรหัสผ่านใหม่, แล้วบันทึกไฟล์ใหม่อีกครั้ง

## ปัญหาที่พบบ่อย & เคล็ดลับ
- **โฟลเดอร์ขนาดใหญ่:** Aspose.Zip ใช้การสตรีมข้อมูล, ทำให้การใช้หน่วยความจำต่ำแม้กับไดเรกทอรีขนาดใหญ่  
- **ความซับซ้อนของรหัสผ่าน:** ใช้รหัสผ่านที่แข็งแรง (ผสมตัวอักษร, ตัวเลข, สัญลักษณ์) เพื่อเพิ่มความปลอดภัย  
- **ข้อผิดพลาดเรื่องลิขสิทธิ์:** ตรวจสอบว่าคุณได้ใส่ไฟล์ลิขสิทธิ์ที่ถูกต้อง; หากไม่เช่นนั้นไลบรารีจะทำงานในโหมดประเมินผลพร้อมข้อจำกัด
