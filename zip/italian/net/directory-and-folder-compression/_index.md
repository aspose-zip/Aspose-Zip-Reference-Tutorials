## Problemi comuni e consigli

- **File di grandi dimensioni:** Quando lavori con file superiori a 2 GB, abilita la modalità **Zip64** per evitare limitazioni di dimensione.

## Domande frequenti

**D: Posso creare un archivio zip protetto da password usando Aspose.Zip?**  
R: Sì. Quando salvi l'archivio, puoi fornire un `ZipPassword` e scegliere un algoritmo di crittografia come AES‑256.

**D: Aspose.Zip supporta lo streaming di file di grandi dimensioni senza caricarli interamente in memoria?**  
R: Assolutamente. Puoi lavorare con oggetti `FileStream`, consentendo di comprimere o estrarre file di qualsiasi dimensione in modo efficiente.

**D: Cosa succede se devo dividere un grande archivio in più parti?**  
R: Usa il metodo `SplitArchive` per definire una dimensione massima per parte; Aspose.Zip creerà automaticamente file divisi sequenziali.

**D: È possibile aggiungere file a un archivio zip esistente?**  
R: Sì. Apri l'archivio in modalità `Update` e chiama `AddFile` o `AddFolder` per aggiungere nuovo contenuto.

**D: Quali runtime .NET sono ufficialmente supportati?**  
R: Aspose.Zip for .NET supporta .NET Framework 4.5+, .NET Core 3.1+, e .NET 5/6/7+.
