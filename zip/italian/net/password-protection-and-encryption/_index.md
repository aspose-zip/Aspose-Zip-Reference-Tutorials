## Domande frequenti

**Q: Come aggiungo una password ai file zip usando Aspose.Zip?**  
A: Usa la classe `ZipArchive`, imposta la proprietà `Password` e scegli un metodo di crittografia come AES‑256.

**Q: Posso proteggere con password una directory senza comprimerla?**  
A: Sì, Aspose.Zip consente di creare un archivio che contiene una struttura di cartelle e applicare una password all'intero archivio.

**Q: Qual è la differenza tra “encrypt zip with aes” e la protezione con password tradizionale?**  
A: La crittografia AES fornisce una sicurezza crittografica forte (chiavi da 128/256 bit), mentre le password ZIP tradizionali usano ZipCrypto più debole.

**Q: Come decomprimo archivi zip crittografati AES in .NET?**  
A: Chiama `ZipArchive.ExtractAll` (o `ExtractEntry`) e fornisci la stessa password usata durante la creazione dell'archivio.

**Q: È possibile estrarre flussi di file crittografati AES senza scrivere su disco?**  
A: Sì, Aspose.Zip supporta l'estrazione in‑memoria lavorando direttamente con gli stream.
