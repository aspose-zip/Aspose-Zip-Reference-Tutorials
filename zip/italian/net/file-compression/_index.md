## Domande frequenti

**D: Posso usare Aspose.Zip per creare un zip archive .net su .NET Core?**  
R: Assolutamente. Aspose.Zip supporta pienamente .NET Core 3.1 e versioni successive, così come .NET 5/6/7.

**D: Come posso modificare un archivio esistente senza estrarlo prima?**  
R: Usa le API “modify zip file c#” – puoi aggiungere, eliminare o rinominare le voci direttamente sullo stream dell'archivio.

**D: Quale metodo di compressione offre il miglior rapporto per i file di testo?**  
R: Generalmente, Bzip2 o LZMA forniscono una compressione maggiore per il testo, ma sono più lenti rispetto a Deflate. Provali entrambi per trovare il giusto equilibrio.

**D: Esiste un modo per crittografare i file durante la creazione dell'archivio zip?**  
R: Sì, Aspose.Zip supporta la crittografia AES. Puoi impostare una password sull'oggetto ZipArchive prima di salvare.

**D: Devo preoccuparmi della codifica dei nomi file?**  
R: Aspose.Zip gestisce automaticamente i nomi file Unicode, ma puoi impostare esplicitamente la codifica se hai bisogno di compatibilità con sistemi legacy.
