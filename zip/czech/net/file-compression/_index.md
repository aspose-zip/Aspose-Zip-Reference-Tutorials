## Často kladené otázky

**Q: Mohu použít Aspose.Zip k vytvoření zip archivu .net na .NET Core?**  
A: Rozhodně. Aspose.Zip plně podporuje .NET Core 3.1 a novější, stejně jako .NET 5/6/7.

**Q: Jak mohu upravit existující archiv, aniž bych ho nejprve rozbalil?**  
A: Použijte API “modify zip file c#” – můžete přidávat, mazat nebo přejmenovávat položky přímo ve streamu archivu.

**Q: Která metoda komprese poskytuje nejlepší poměr pro textové soubory?**  
A: Obecně Bzip2 nebo LZMA poskytují vyšší kompresi pro text, ale jsou pomalejší než Deflate. Otestujte obě a najděte správnou rovnováhu.

**Q: Existuje způsob, jak šifrovat soubory při vytváření zip archivu?**  
A: Ano, Aspose.Zip podporuje AES šifrování. Můžete nastavit heslo na objekt ZipArchive před uložením.

**Q: Musím se starat o kódování názvů souborů?**  
A: Aspose.Zip automaticky zpracovává Unicode názvy souborů, ale můžete explicitně nastavit kódování, pokud potřebujete kompatibilitu se staršími systémy.
